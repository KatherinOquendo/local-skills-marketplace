# Deliverables

Process outputs, discovery artifacts, architecture documents, and proposals organized by entity.

| Entity | Scope | Examples |
|--------|-------|---------|
| `sofka/` | Sofka client deliverables | Discovery briefs, AS-IS analyses, solution roadmaps, executive pitches |
| `metodologia/` | MetodologIA deliverables | Methodology documentation, community guides, training materials |
| `javier-montano/` | Personal deliverables | Conference talks, blog posts, personal project documentation |

## Subdirectories per Entity

| Directory | Content |
|-----------|---------|
| `discovery/` | Discovery pipeline outputs (briefs, AS-IS, scenarios, roadmaps) |
| `architecture/` | Architecture decision records, C4 diagrams, technical specs |
| `proposals/` | Business proposals, project pitches, investment cases |
