# Governance

Policies, standards, frameworks, and templates organized by entity.

| Entity | Scope | Examples |
|--------|-------|---------|
| `sofka/` | Sofka Technologies enterprise governance | Discovery process standards, quality gates, client engagement policies |
| `metodologia/` | MetodologIA community governance | Open methodology contribution guidelines, skill quality standards, GPL compliance |
| `javier-montano/` | Personal professional governance | Personal development frameworks, career policies, knowledge management |

## Subdirectories per Entity

| Directory | Content |
|-----------|---------|
| `policies/` | Rules and constraints that must be followed |
| `standards/` | Measurable quality bars and conventions |
| `frameworks/` | Reusable decision-making structures |
| `templates/` | Starter documents for recurring governance artifacts |
