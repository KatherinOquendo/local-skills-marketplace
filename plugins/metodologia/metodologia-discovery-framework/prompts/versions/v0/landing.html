Arsenal de Skills

Comité de Expertos

48 agentes especializados.

12 agentes core que gobiernan el pipeline + 36 especialistas de dominio que se activan según el tipo de servicio y la fase. Haz clic en cualquier agente para ver su rol.

12 Core (siempre activos)
36 Especialistas de dominio

Cada agente tiene su constitución: rol, herramientas, restricciones, y criterios de activación. El discovery-conductor orquesta el comité.

Comenzar

3 pasos para activar tu ecosistema.

Instala el plugin en Claude Code y convierte tu terminal en el consultor técnico más riguroso de MetodologIA.

Paso 01
Clonar el repositorio
Clona el repo en tu directorio de plugins de Claude Code.
git clone https://github.com/javiermontano-metodologia/metodologia-discovery-framework.git
~/skills/plugins/metodologia-discovery-framework
Paso 02
Configurar CLAUDE.md
Copia el CLAUDE.md del repo a tu configuración global para activar el enrutamiento de comandos /mao:.
# Lee el CLAUDE.md del repo
# para instrucciones completas
Paso 03
Ejecutar tu primer discovery
Usa el comando express para validar la instalación con un Go/No-Go rápido.
/mao:run-express
