## Portafolio de servicios QA

| Servicio | Descripcion | Entregables tipicos |
|----------|-------------|---------------------|
| **Test Strategy** | Diseno de estrategia de testing alineada a riesgos | Test strategy doc, coverage plan |
| **Test Automation** | Automatizacion de pruebas (UI, API, performance) | Automation framework, scripts |
| **PITT** | Equipos Independientes de Testing (separacion dev/QA) | Test reports, defect analysis |
| **Testing Factory** | Fabrica de testing con SLAs definidos | Continuous testing, dashboards |
| **Strategic QA Consulting** | Evaluacion de madurez QA, TMMi assessment | Maturity report, roadmap |
| **Performance Testing** | Load, stress, endurance testing | Performance benchmarks, bottleneck analysis |
| **Security Testing** | SAST, DAST, penetration testing | Vulnerability reports |
