## MLOps maturity

| Nivel | Descripcion |
|-------|-------------|
| **0 - Manual** | Notebooks, deployment manual, sin monitoring |
| **1 - Automated training** | Pipeline de entrenamiento, deployment semi-automatico |
| **2 - Automated deployment** | CI/CD para modelos, A/B testing automatico |
| **3 - Full MLOps** | Feature store, model registry, drift detection, retraining automatico |
