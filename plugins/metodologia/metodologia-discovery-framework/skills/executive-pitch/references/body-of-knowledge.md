## Glosario

| Término | Definición |
|---------|-----------|
| **NPV (Net Present Value)** | Suma de flujos de caja futuros descontados al presente menos la inversión inicial. NPV > 0 indica que el proyecto genera valor. |
| **IRR (Internal Rate of Return)** | Tasa de descuento que hace NPV = 0. Se compara contra el hurdle rate de la empresa. |
| **Payback Period** | Meses hasta que los beneficios acumulados igualan la inversión acumulada. |
| **Cost of Inaction (CoI)** | Costo total (directo + indirecto + oportunidad) de no ejecutar el cambio propuesto. Se mide en 4 dimensiones: técnica, operacional, competitiva, regulatoria. |
| **PAS (Problem-Agitate-Solve)** | Framework de persuasión: presentar el problema con datos, agitar con impacto emocional y urgencia, resolver con la propuesta y ROI. |
| **Burn Rate de Inacción** | Costo mensual de mantener el status quo. Se calcula como CoI anual / 12, ajustado por compounding. |
| **Sensitivity Analysis** | Variación de supuestos clave (+/-20% costo, +/-10% beneficio) para evaluar robustez del business case. |
| **TCO (Total Cost of Ownership)** | Costo total de implementación + operación durante el horizonte de evaluación (típicamente 3 años). |
| **WSJF (Weighted Shortest Job First)** | Priorización SAFe: (Business Value + Time Criticality + Risk Reduction) / Job Size. |
| **FTE-months** | Unidad de esfuerzo: 1 persona a tiempo completo durante 1 mes. MetodologIA usa FTE-months en lugar de precios. |
| **Hurdle Rate** | Tasa mínima de retorno que un proyecto debe superar para ser aprobado. Típicamente 10-15% para enterprise tech. |
| **Strangler Fig** | Patrón de migración donde el sistema nuevo envuelve progresivamente al legacy hasta reemplazarlo. |
