# Copywriting — Ejemplos de Referencia

Este directorio contiene ejemplos de alta calidad que sirven como benchmark para el skill de copywriting.

## Contenido

| Archivo | Descripción |
|---------|-------------|
| `sample-output.md` | Extracto de pitch ejecutivo en markdown — cliente ficticio RetailCo |
| `sample-output.html` | Versión HTML con MetodologIA Design System (misma pieza) |

## Caso de Ejemplo

**Cliente ficticio:** RetailCo — e-commerce con plataforma monolítica legacy.

**Pieza:** Secciones clave de un pitch ejecutivo: resumen ejecutivo, diagnóstico del problema, costo de inacción, escenarios comparados, propuesta de valor y call to action.

**Técnicas demostradas:**
- TL;DR con 5 bullets accionables
- Problem-Agitate-Solve (PAS) en la sección de diagnóstico
- Cost-of-Inaction (COI) con proyección trimestral
- Evidence tagging completo (`[CÓDIGO]`, `[CONFIG]`, `[DOC]`, `[STAKEHOLDER]`, `[INFERENCIA]`)
- Números siempre con contexto comparativo (vs baseline, vs benchmark)
- Calibración de tono para audiencia mixta (CEO + CTO)
- Call to Action con acción + plazo + next step + consecuencia de no actuar
- Zero muletillas
- Métricas en FTE-meses (nunca precios)

## Uso como Benchmark

El editorial committee debe usar estos ejemplos como referencia de calidad mínima. Todo copy producido por el skill debe igualar o superar este nivel en:

1. Densidad de evidencia (100% de claims tagueados)
2. Concisión (0 muletillas, oraciones < 25 palabras promedio)
3. Accionabilidad (CTAs con timeline + next step)
4. Calibración por audiencia
