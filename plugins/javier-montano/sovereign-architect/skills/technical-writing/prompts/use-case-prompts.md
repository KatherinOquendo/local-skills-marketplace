# Technical Writing — Use Case Prompts

## Prompt 1 — Architecture Assessment Document

> Write a technical assessment document for a microservices architecture review.
> Apply progressive disclosure with three reading levels. Tag all findings with
> evidence types. Include a TL;DR, executive summary, detailed findings, and
> appendix with raw data.

## Prompt 2 — API Migration Guide

> Produce a migration guide for teams moving from REST API v2 to v3.
> Structure for random access (teams will look up specific endpoints).
> Include breaking changes, migration steps per endpoint, and rollback procedures.
> Use decision tables for conditional migration paths.

## Prompt 3 — Post-Incident Technical Report

> Write a post-incident report for a 4-hour production outage caused by
> a database connection pool exhaustion. Apply the claim-evidence-implication
> pattern. Include timeline, root cause analysis, contributing factors,
> and remediation plan with evidence tags on every factual assertion.
