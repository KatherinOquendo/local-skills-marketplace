# Cloud Migration — Use Case Prompts

## Prompt 1 — 7R Portfolio Assessment
> "Analyze this application portfolio and classify each workload using the 7R migration framework. For each application, assess technology stack, dependencies, SLAs, and data volumes. Produce a migration disposition matrix with strategy recommendation and complexity score."

## Prompt 2 — Wave Planning
> "Given this set of applications and their dependency map, create a migration wave plan. Group workloads into waves, sequence by dependency and risk, define success criteria per wave, and estimate timeline. Ensure no wave creates orphaned dependencies."

## Prompt 3 — Cutover Runbook
> "Design a cutover runbook for migrating this database-backed application from on-premises to cloud. Include data sync strategy, DNS switchover steps, health check validation, rollback triggers, and a minute-by-minute timeline for the cutover window."
