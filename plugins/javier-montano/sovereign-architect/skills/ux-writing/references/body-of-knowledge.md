# UX Writing — Body of Knowledge

## Core Concepts

1. **Cognitive Load Theory**: Readers have limited working memory. Document design should minimize extraneous load and optimize germane load (the load that contributes to understanding).
2. **Scanability**: The ability to extract key information without reading the full text. Achieved through headings, lists, bold keywords, and front-loaded sentences.
3. **Information Scent**: The degree to which navigation elements (headings, links, TOC) signal to readers that they are heading toward the content they need.
4. **Readability Metrics**: Quantitative measures of text difficulty. Flesch-Kincaid, Gunning Fog, and SMOG index each emphasize different aspects of complexity.
5. **Semantic Heading Hierarchy**: Using heading levels (H1-H6) to represent logical containment relationships, not visual size preferences. Critical for screen readers and navigation.
6. **Plain Language**: Writing that uses simple, direct language without sacrificing technical precision. Not dumbing down — optimizing for comprehension speed.

## Patterns

1. **F-Pattern Layout**: Web readers scan in an F-shape — two horizontal stripes across the top, then a vertical stripe down the left. Place key information along this pattern.
2. **Chunking**: Breaking content into discrete, manageable units of 3-7 items. Lists, tables, and short paragraphs apply chunking to prose.
3. **Signposting**: Explicit markers that tell readers where they are, where they have been, and where they are going (e.g., "In this section...", "As discussed in Section 3...").
4. **Dual Coding**: Presenting information in both verbal and visual forms. Diagrams paired with descriptions improve retention and comprehension.

## Tools

1. **Hemingway Editor**: Readability analysis tool that highlights complex sentences, passive voice, and adverb overuse.
2. **axe/WAVE**: Accessibility testing tools for heading hierarchy and WCAG compliance.
3. **Mermaid.js**: Diagram-as-code for visual elements that travel with the document source.
4. **Grammarly/LanguageTool**: Automated writing assistance for grammar, clarity, and tone.
