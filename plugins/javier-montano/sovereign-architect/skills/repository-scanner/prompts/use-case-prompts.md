# Repository Scanner — Use Case Prompts

## Prompt 1: Full Stack Scan
> "Scan this repository and tell me every technology, framework, and language being used. Include versions where possible."

## Prompt 2: Entry Point Discovery
> "Where are the entry points in this codebase? I need to know how to start the application and what services it exposes."

## Prompt 3: Monorepo Mapping
> "This looks like a monorepo. Map out all the packages, their dependencies on each other, and what each one does."
