# Scenario Analysis — Use Case Prompts

## Prompt 1 — Technology Selection
> "Compare these technology options for our new project: [Option A] vs [Option B] vs [Option C]. Define evaluation criteria, weight by our priorities, score each option with evidence, perform sensitivity analysis, and produce a decision matrix with recommendation."

## Prompt 2 — Architecture Trade-Off Analysis
> "Analyze the trade-offs between these architectural approaches: monolith, modular monolith, and microservices for this project. Score across 6 dimensions (feasibility, cost, time, risk, team, strategy), visualize the decision tree, and produce an ADR for the recommendation."

## Prompt 3 — Build vs. Buy Analysis
> "Evaluate build vs. buy for this capability. Compare custom development against 3 vendor solutions. Include TCO analysis (3-year), customization requirements, integration complexity, vendor risk, and team impact. Produce a weighted decision matrix."
