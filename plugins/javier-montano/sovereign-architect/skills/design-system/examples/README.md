# Design System — Examples

Sample outputs demonstrating the design system skill at moat level.

## Contents

| File | Description |
|------|-------------|
| `sample-output.md` | Design token configuration and a styled section in Markdown format |
| `sample-output.html` | Full HTML page demonstrating design system components — headers, tables, callouts, cards, Mermaid integration |

## Usage

- `sample-output.md` — reference for `FORMATO=markdown` output. Shows token documentation and component usage in Markdown tables and code blocks.
- `sample-output.html` — reference for `FORMATO=html` output. Open in any browser to see the rendered design system with brand tokens applied.

## Notes

- Both samples use the **neutral default** brand config (blue primary). Replace `:root` values with any `brand-config.json` to see brand-specific rendering.
- The HTML sample includes print stylesheet, skip-to-content link, and WCAG AA compliant contrast ratios.
