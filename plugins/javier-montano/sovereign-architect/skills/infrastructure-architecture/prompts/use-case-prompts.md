# Infrastructure Architecture — Use Case Prompts

## Prompt 1 — Network Topology Design
> "Design the network architecture for a multi-account cloud deployment. Define VPC topology, CIDR allocation, subnet strategy, connectivity (VPN/peering/transit gateway), and security controls. Produce a network diagram and Terraform module structure."

## Prompt 2 — HA/DR Strategy
> "Design the high availability and disaster recovery strategy for this production system. Define RTO/RPO per workload tier, multi-AZ deployment topology, failover mechanisms, backup strategy, and DR runbook. Validate with failure scenario analysis."

## Prompt 3 — IaC Audit
> "Audit the Terraform/IaC codebase for this infrastructure. Check for security misconfigurations, drift risk, missing state locking, hardcoded values, and compliance gaps. Produce a findings report with severity ratings and remediation steps."
