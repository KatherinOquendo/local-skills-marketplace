# Data Model Analyzer — Use Case Prompts

## Prompt 1: Schema Audit
> "Map out the entire data model — tables, relationships, and indexes. Generate an ER diagram."

## Prompt 2: Migration Health
> "Review the migration history. Are there any problematic patterns or risky migrations?"

## Prompt 3: Normalization Review
> "Is our schema properly normalized? Where have we denormalized and is it justified?"
