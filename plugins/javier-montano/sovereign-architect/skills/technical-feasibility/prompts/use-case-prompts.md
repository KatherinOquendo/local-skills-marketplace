# Use Case Prompts — Technical Feasibility

## 1. Full Feasibility Assessment
```
Validate technical feasibility of {PROPOSAL} for {PROJECT}.
Assess across 7 dimensions: architecture, performance, security, scalability, cost, timeline, team.
Deliver scored matrix with go/no-go recommendation.
```

## 2. Claim Fact-Checking
```
Fact-check these technical claims from {SOURCE}:
{CLAIMS_LIST}
Tag each as [HECHO], [INFERENCIA], or [SUPUESTO] with evidence.
```

## 3. Quick Feasibility Check
```
Quick feasibility check: can we {PROPOSED_CHANGE} in {TIMELINE}?
Focus on blockers and showstoppers only.
Deliver 1-page verdict with top risks.
```
