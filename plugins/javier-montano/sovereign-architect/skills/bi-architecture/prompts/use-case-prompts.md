# BI Architecture — Use Case Prompts

## Prompt 1 — Semantic Layer Design
> "Design a semantic layer for this data warehouse. Define the metrics catalog (revenue, churn, engagement, etc.), dimension hierarchies, and calculation logic. Specify how the semantic layer should be implemented (Looker LookML, Cube.js, MetricFlow) and how metrics are governed."

## Prompt 2 — Dashboard Architecture Review
> "Review the dashboard architecture for this BI platform. Assess dashboard taxonomy, metric consistency, query performance, filter design, and refresh strategy. Identify dashboards with conflicting metric definitions and produce a consolidation plan."

## Prompt 3 — KPI Framework Design
> "Design a KPI framework for this SaaS business aligned with OKRs. Define strategic, tactical, and operational KPIs, their calculation formulas, data sources, refresh frequency, and alerting thresholds. Produce a KPI hierarchy diagram and executive scorecard specification."
