# Dependency Auditor — Use Case Prompts

## Prompt 1: Security Audit
> "Check all dependencies for known vulnerabilities. Give me a prioritized list of what needs patching immediately."

## Prompt 2: License Compliance
> "We're shipping this as a commercial product. Are there any copyleft or problematic licenses in our dependency tree?"

## Prompt 3: Staleness Report
> "How outdated are our dependencies? Which ones are abandoned and need replacement?"
