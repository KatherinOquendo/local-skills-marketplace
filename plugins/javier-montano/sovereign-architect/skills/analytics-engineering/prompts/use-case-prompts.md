# Analytics Engineering — Use Case Prompts

## Prompt 1 — Dimensional Model Design
> "Design a dimensional model for this e-commerce database. Identify fact tables (orders, events, payments), dimension tables (customers, products, time, geography), and define the grain, measures, and SCD strategy for each. Produce an ER diagram and dbt model skeleton."

## Prompt 2 — dbt Project Review
> "Review this dbt project for best practices. Check model layering (staging/intermediate/mart), naming conventions, test coverage, documentation completeness, materialization choices, and incremental model correctness. Produce a findings report with severity ratings."

## Prompt 3 — Metrics Layer Design
> "Design a metrics layer for this analytics platform. Define key business metrics (revenue, churn, LTV, conversion rate) with their SQL calculations, dimensions for slicing, and time grains. Specify how metrics should be served to BI tools and embedded analytics."
