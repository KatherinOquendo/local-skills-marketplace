# Diagnostic Engine — Use Case Prompts

## Prompt 1: Production Performance Degradation
> "Our API response times have increased 3x over the past month but we haven't deployed any major changes. Diagnose the root cause by examining the codebase, database queries, and infrastructure configuration. Classify every finding with evidence tags."

## Prompt 2: Codebase Health Assessment
> "We inherited this codebase from another team 6 months ago. Before we plan any new features, diagnose its overall health: technical debt, security concerns, testing gaps, and architectural risks. Prioritize findings by severity and blast radius."

## Prompt 3: Post-Incident Root Cause Analysis
> "Last Friday we had a 2-hour outage caused by a database connection pool exhaustion. The immediate fix was restarting the service. Diagnose the underlying cause, identify what made this possible, and find any similar time-bomb patterns elsewhere in the codebase."
