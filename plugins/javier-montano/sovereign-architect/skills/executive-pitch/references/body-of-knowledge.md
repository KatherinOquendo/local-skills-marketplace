# Executive Pitch — Body of Knowledge

## Core Concepts

1. **Value Pillar**: A strategic theme that connects a technical capability to a measurable business outcome. Pillars are the structural units of an executive pitch.
2. **Cost of Inaction**: The quantified impact of maintaining the status quo. Often the most compelling argument in an executive pitch.
3. **Return on Investment (ROI)**: The ratio of net benefit to total investment, expressed as a percentage. Must include time dimension (when returns materialize).
4. **Payback Period**: The time required for cumulative benefits to equal cumulative investment. A key metric for risk-averse decision-makers.
5. **Decision Framework**: A structured model that guides the audience through the evaluation criteria and leads logically to the recommendation.
6. **Confidence Interval**: The range within which a projected value is expected to fall, given stated assumptions. Communicates uncertainty honestly.

## Patterns

1. **Pyramid Principle**: Start with the answer, then support with arguments, then evidence. Barbara Minto's framework for structured communication.
2. **Before/After/Bridge**: Describe the current state (pain), the desired state (outcome), and how the recommendation bridges the gap.
3. **Three-Act Structure**: Setup (context and problem), confrontation (options and analysis), resolution (recommendation and next steps).
4. **Proof by Analogy**: Reference comparable implementations in similar organizations to build credibility when direct evidence is unavailable.

## Tools

1. **Mermaid Gantt Charts**: Timeline visualizations for investment and return projections embedded in Markdown.
2. **SWOT Analysis**: Strengths, Weaknesses, Opportunities, Threats framework for positioning the recommendation.
3. **Business Model Canvas**: Visual framework for articulating value propositions in business terms.
4. **Impact/Effort Matrix**: 2x2 grid for prioritizing value pillars by business impact and implementation effort.
