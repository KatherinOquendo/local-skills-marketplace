# Solution Roadmap — Use Case Prompts

## Prompt 1 — Monolith Decomposition Roadmap

> Create a phased roadmap for decomposing a monolithic e-commerce platform into
> microservices. Map across three investment horizons. Phase 1 must deliver value
> within 3 months. Include dependency analysis, team allocation, and risk-adjusted
> timelines with confidence intervals.

## Prompt 2 — Cloud Migration Roadmap

> Design the execution roadmap for migrating 40 applications from on-premise to AWS.
> Categorize applications by migration strategy (rehost, replatform, refactor).
> Sequence phases to maximize early wins while managing operational risk.
> Include decision points for re-evaluating the approach after each wave.

## Prompt 3 — Developer Platform Roadmap

> Build a 12-month roadmap for establishing an internal developer platform.
> Sequence: CI/CD standardization, container orchestration, observability stack,
> and self-service infrastructure. Each phase must show measurable DORA metric
> improvement. Include the dual-track discovery/delivery schedule.
