# Copywriting — Body of Knowledge

## Core Concepts

1. **Value Proposition**: A clear statement of the outcome a solution delivers, for whom, and why it is uniquely better than alternatives. The foundation of all persuasive copy.
2. **Call to Action (CTA)**: A specific, actionable next step the reader should take. Effective CTAs are concrete, time-bound, and low-friction.
3. **Pain-Gain Framing**: Structuring copy around the contrast between the current state (problems, costs, risks) and the future state (outcomes, savings, capabilities).
4. **Social Proof**: Evidence that others have achieved the promised outcomes. Case studies, metrics, and testimonials reduce perceived risk.
5. **Cognitive Fluency**: Readers trust and prefer content that is easy to process. Simple language, short sentences, and familiar structure increase fluency.
6. **Feature-Benefit Translation**: Converting technical features into reader-relevant benefits. "Event-driven architecture" becomes "real-time data processing that eliminates batch delays."

## Patterns

1. **Problem-Agitate-Solve (PAS)**: State the problem, amplify its impact, then present the solution. Effective for executive summaries.
2. **Before-After-Bridge (BAB)**: Describe the current state, paint the desired future, then show how to get there. Works well for value propositions.
3. **AIDA (Attention-Interest-Desire-Action)**: Sequence for longer copy that moves from awareness to conversion.
4. **One-Sentence Pitch**: Distill the entire value proposition into a single sentence using the formula: "We help [audience] achieve [outcome] by [mechanism]."

## Tools

1. **Hemingway Editor**: Readability and complexity analysis for copy.
2. **CoSchedule Headline Analyzer**: Scoring tool for headline effectiveness.
3. **Value Proposition Canvas**: Visual tool for mapping customer jobs, pains, and gains to product features and benefits.
4. **A/B Testing Frameworks**: Methods for testing copy variations against measurable outcomes.
