# Performance Engineering — Use Case Prompts

## Prompt 1 — Load Test Design
> "Design a load testing strategy for this application. Define test scenarios (baseline, stress, spike, soak), create realistic traffic patterns, specify success criteria per scenario, and design the test infrastructure. Include k6/Gatling script structure."

## Prompt 2 — Bottleneck Analysis
> "Analyze the performance of this application. Review database queries, API endpoint latencies, connection pool configurations, and resource utilization patterns. Identify the top 5 bottlenecks with profiling evidence and prioritize optimization recommendations."

## Prompt 3 — Caching Architecture
> "Design a caching architecture for this read-heavy application. Identify cacheable data, select caching layers (CDN, application, database), define TTL and invalidation strategies, and estimate cache hit ratio improvements. Include cache warming and fallback procedures."
