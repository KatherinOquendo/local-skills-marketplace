# Enterprise Architecture — Use Case Prompts

## Prompt 1 — Capability Map from Codebase
> "Analyze this monorepo and produce a business capability map (L0-L2). For each capability, identify the implementing services, their maturity level, and any gaps or redundancies. Output as a Mermaid diagram with a supporting table."

## Prompt 2 — Technology Radar Assessment
> "Scan the repository for all technologies in use (languages, frameworks, libraries, databases, infrastructure tools). Classify each into Adopt/Trial/Assess/Hold rings with rationale based on community health, maintenance burden, and strategic alignment. Flag any technologies that should be urgently migrated."

## Prompt 3 — Strategic Roadmap from Current State
> "Given the current architecture (see attached AS-IS diagrams and ADRs), produce a 12-month strategic roadmap. Identify the top 5 architectural initiatives, sequence them by dependency and risk, estimate effort in T-shirt sizes, and define quality gates between phases."
