# Data Governance — Use Case Prompts

## Prompt 1 — Data Catalog Design
> "Design a data catalog architecture for this data platform. Define metadata schema, tagging taxonomy, automated discovery pipelines, and search/discovery interface requirements. Include ownership assignment strategy and quality score integration."

## Prompt 2 — PII Discovery & Classification
> "Scan this database schema for PII fields. Classify each column by sensitivity tier (public, internal, confidential, restricted), identify GDPR-relevant data, and produce a data classification report with masking recommendations for non-production environments."

## Prompt 3 — Retention Policy Framework
> "Design a data retention policy framework for this platform. Define retention periods per data classification, implement automated lifecycle management rules, and build deletion audit trails. Map policies to GDPR/CCPA/LGPD requirements."
