# Solutions Architecture — Use Case Prompts

## Prompt 1 — Integration Blueprint
> "Design the integration architecture for this multi-service system. Identify all integration points, select appropriate patterns (sync/async/batch), define data contracts, and specify error handling strategies. Produce a C4 Container diagram and integration matrix."

## Prompt 2 — Cross-Cutting Concerns Audit
> "Audit this codebase for cross-cutting concern implementation. Check how authentication, logging, tracing, resilience, and configuration are handled across services. Identify inconsistencies and produce a unified cross-cutting strategy document."

## Prompt 3 — Solution Design Review
> "Review this solution architecture against its stated NFRs. Walk through the 3 most critical user journeys, identify failure modes, validate scalability assumptions with back-of-envelope calculations, and produce a risk register with mitigations."
