# Input Analysis — Use Case Prompts

## Prompt 1 — RFP Interpretation

> Analyze this RFP document for a banking core modernization project. Identify
> contradictions between stated requirements, surface unstated assumptions about
> the current architecture, and amplify the intent behind each requirement section.
> Produce clarifying questions prioritized by impact on proposal quality.

## Prompt 2 — Stakeholder Interview Synthesis

> Process transcripts from 5 stakeholder interviews about a platform migration.
> Detect contradictions between stakeholder perspectives, identify consensus areas,
> and flag topics where stakeholders have different implicit assumptions.
> Produce a unified interpretation with confidence scores.

## Prompt 3 — Technical Brief Enhancement

> Analyze a 2-page technical brief requesting a microservices architecture assessment.
> The brief is vague on scope and success criteria. Apply intent amplification to
> determine what the requestor likely needs, identify missing context, and produce
> an enriched brief with explicit assumptions and clarifying questions.
