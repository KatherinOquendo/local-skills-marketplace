# Discovery Orchestrator — Example Outputs

Sample outputs demonstrating what the discovery-orchestrator skill produces: the discovery plan, expert committee declaration, and pipeline status reports for a fictional "Acme Corp Banking Modernization" project.

## Files

- `sample-output.md` — Markdown discovery plan (default output)
- `sample-output.html` — HTML branded version

## Usage

These examples show the initialization protocol output (Phase -1). The orchestrator produces status reports after each phase and gate evaluation reports at each quality gate.
