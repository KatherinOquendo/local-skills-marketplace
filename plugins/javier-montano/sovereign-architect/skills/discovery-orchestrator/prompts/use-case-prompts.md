# Discovery Orchestrator — Use Case Prompts

## Prompt 1 — Full Platform Assessment

> Orchestrate a complete technical discovery for a SaaS platform migration.
> Sequence the pipeline: codebase analysis, architecture assessment, flow mapping,
> risk identification, and roadmap generation. Define gates between phases
> and produce the dependency graph.

## Prompt 2 — Focused Security Discovery

> Run a security-focused discovery pipeline: threat modeling, dependency audit,
> compliance gap analysis, and remediation roadmap. The scope is a fintech
> application handling PCI-DSS regulated data. Time-box each phase to 2 hours
> and define the minimum viable output for each.

## Prompt 3 — Legacy System Modernization Discovery

> Orchestrate a modernization discovery for a 15-year-old Java monolith.
> Include: current state mapping, domain boundary identification, strangler fig
> candidate analysis, and phased migration roadmap. Identify which analyses
> can run in parallel and which have hard dependencies.
