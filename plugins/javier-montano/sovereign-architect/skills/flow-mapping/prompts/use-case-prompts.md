# Flow Mapping — Use Case Prompts

## Prompt 1 — End-to-End Order Flow

> Map the complete order processing flow from customer placement through fulfillment.
> Include all service boundaries, async messaging, payment gateway interactions,
> and inventory reservation steps. Produce a Mermaid sequence diagram and flag
> any single points of failure.

## Prompt 2 — Authentication and Authorization Flow

> Trace the authentication flow from login request through token issuance,
> including SSO federation, MFA challenges, and session management.
> Identify all external identity provider integrations and document
> the token lifecycle (issuance, refresh, revocation).

## Prompt 3 — Data Pipeline Integration Map

> Map all data ingestion, transformation, and delivery flows across the
> analytics pipeline. Include source systems, ETL/ELT stages, intermediate
> storage, and downstream consumers. Highlight data freshness guarantees
> and identify flows without error handling or retry mechanisms.
