# Stakeholder Mapping — Use Case Prompts

## Prompt 1 — Stakeholder Analysis
> "Map the stakeholders for this technical project. Identify all relevant parties (sponsors, decision makers, teams, end users), create an influence/interest matrix, and design engagement strategies per quadrant. Include hidden stakeholders often missed."

## Prompt 2 — RACI Matrix
> "Create a RACI matrix for this project's major deliverables and decisions. Identify the Responsible, Accountable, Consulted, and Informed parties for each item. Validate that no deliverable has multiple Accountable people and no person is overloaded."

## Prompt 3 — Communication Plan
> "Design a communication plan for this engineering initiative. Define channels, frequency, and artifacts per stakeholder group. Include escalation paths for blocked decisions and feedback mechanisms for stakeholder input."
