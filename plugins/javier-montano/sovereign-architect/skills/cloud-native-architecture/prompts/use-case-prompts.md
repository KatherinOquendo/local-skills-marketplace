# Cloud-Native Architecture — Use Case Prompts

## Prompt 1 — Cloud-Native Readiness Assessment
> "Assess this application's cloud-native readiness. Check 12-factor compliance, containerization patterns, Kubernetes manifests, CI/CD pipeline, and observability stack. Score each dimension 1-5 on the CNMM scale and produce a remediation plan for gaps."

## Prompt 2 — Kubernetes Topology Design
> "Design the Kubernetes deployment topology for this microservices system. Define namespaces, resource quotas, network policies, autoscaling configuration, and storage strategy. Produce YAML manifests and a deployment architecture diagram."

## Prompt 3 — FinOps Cost Optimization
> "Analyze the Kubernetes resource configurations in this repo (requests, limits, HPA settings). Identify over-provisioned workloads, right-sizing opportunities, and spot instance candidates. Estimate monthly savings and produce an optimization plan."
