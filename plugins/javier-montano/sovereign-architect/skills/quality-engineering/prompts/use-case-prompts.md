# Quality Engineering — Use Case Prompts

## Prompt 1 — Quality Gates Design
> "Design quality gates for this CI/CD pipeline. Define criteria for each stage (commit, PR, merge, staging, production), specify thresholds, implement automated enforcement, and design exception handling. Include gate effectiveness metrics."

## Prompt 2 — Quality Metrics Dashboard
> "Design a quality metrics dashboard for this engineering organization. Define leading and lagging indicators, specify data sources, design team-level and org-level views, and establish alerting thresholds for quality degradation trends."

## Prompt 3 — Quality Engineering Assessment
> "Assess the quality engineering practices of this project. Evaluate test coverage, automation maturity, quality gate effectiveness, defect patterns, and release quality. Produce a maturity score and improvement roadmap."
