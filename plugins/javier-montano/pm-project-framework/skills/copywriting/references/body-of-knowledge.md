# Body of Knowledge — Copywriting

## 1. Foundations
PM copywriting crafts clear, persuasive, and audience-appropriate written communications for project artifacts: charters, status reports, stakeholder presentations, executive summaries, and change requests. [DOC]

### 1.1 Writing Frameworks for PM
| Framework | Application | Structure |
|-----------|-----------|-----------|
| BLUF (Bottom Line Up Front) | Executive communications | Key message first, then details [STAKEHOLDER] |
| Pyramid Principle (Minto) | Reports and analyses | Conclusion then supporting arguments [DOC] |
| STAR (Situation-Task-Action-Result) | Status updates, case studies | Contextual storytelling [PLAN] |
| SCR (Situation-Complication-Resolution) | Problem statements, escalations | Narrative persuasion [STAKEHOLDER] |

### 1.2 Tone by Audience
| Audience | Tone | Detail Level | Action Orientation |
|----------|------|-------------|-------------------|
| C-suite | Strategic, concise | High-level only | Decisions needed [STAKEHOLDER] |
| Steering Committee | Formal, evidence-based | Summary + drill-down | Approvals needed [PLAN] |
| Team | Direct, collaborative | Detailed | Tasks and ownership [SCHEDULE] |
| Client | Professional, empathetic | Tailored to agreement | Status and next steps [STAKEHOLDER] |

## 2. PM Document Types
| Document | Word Count | Key Quality | Evidence |
|----------|-----------|-------------|----------|
| Executive summary | 200-400 words | Actionable clarity | [STAKEHOLDER] |
| Status report | 500-1000 words | Evidence-backed metrics | [METRIC] |
| Change request | 300-600 words | Clear impact analysis | [PLAN] |
| Risk escalation | 200-400 words | Urgency with facts | [SCHEDULE] |
| Project charter | 1000-2000 words | Comprehensive scope | [DOC] |
