# Metaprompts — Copywriting

## Meta-Strategy 1: So-What Test
**Purpose**: Ensure every sentence serves the reader.
**Strategy**: For each paragraph, ask: "So what? What should the reader do with this information?" If no clear answer, cut or rewrite.
**Application template**:
> Review `{document}` paragraph by paragraph. Apply So-What test. Tag sentences without clear purpose. Rewrite or remove. Target: every paragraph has an actionable takeaway. [STAKEHOLDER]

## Meta-Strategy 2: Evidence-Language Alignment
**Purpose**: Match confidence language to evidence quality.
**Strategy**: Strong claims need [METRIC] evidence. Moderate claims use [INFERENCIA]. Weak claims use [SUPUESTO] with hedging language.
**Application template**:
> Audit `{document}` for evidence-language alignment. Flag strong claims without strong evidence. Soften language for [INFERENCIA] items. Add explicit disclaimers for [SUPUESTO] items. [DOC]

## Meta-Strategy 3: Readability Optimization
**Purpose**: Ensure documents are accessible to their intended audience.
**Strategy**: Target Flesch-Kincaid Grade Level 10-12 for executive documents. Use short sentences (avg 15-20 words). One idea per paragraph. Active voice preferred.
**Application template**:
> Check readability of `{document}`. If grade level >14, simplify sentences and vocabulary. If any paragraph exceeds 5 sentences, split. Ensure all acronyms defined on first use. [DOC]
