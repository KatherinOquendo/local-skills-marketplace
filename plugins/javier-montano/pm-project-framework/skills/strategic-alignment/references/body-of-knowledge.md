# Body of Knowledge — Strategic Alignment & OKR Tracking

> Academic, industrial, and practitioner sources for strategy-to-project traceability, OKR frameworks, and portfolio alignment.

---

## 1. Foundational Frameworks

### PMBOK 7th Edition — Stewardship & Value Delivery

- **Relevance:** PMBOK 7 centers on value delivery. Projects exist to deliver strategic value. The Stewardship principle requires PMs to ensure projects remain aligned with organizational strategy throughout their lifecycle.
- **Key concepts:** Benefits realization, value delivery system, organizational strategy alignment, portfolio management.
- **Use in skill:** The 10-step protocol ensures continuous alignment validation, not just initial justification.
- **Source:** PMI (2021). *PMBOK Guide — 7th Edition*.

### OKR Framework (Doerr / Grove)

- **Relevance:** Objectives and Key Results connect strategic goals to measurable outcomes. OKRs cascade from organizational strategy to team-level execution, providing traceable links from strategy to daily work.
- **Key principles:** Objectives are qualitative and inspiring; Key Results are quantitative and measurable. 3-5 OKRs per level. 70% achievement is healthy (stretch targets).
- **Use in skill:** OKR mapping is the primary mechanism for strategy-to-project traceability.
- **Source:** Doerr, J. (2018). *Measure What Matters*; Grove, A. (1983). *High Output Management*.

### Balanced Scorecard (Kaplan & Norton)

- **Relevance:** Four-perspective strategic framework: Financial, Customer, Internal Process, Learning & Growth. Ensures projects address all strategic dimensions, not just financial returns.
- **Key insight:** Organizations that measure only financial outcomes miss leading indicators of future performance.
- **Use in skill:** Use Balanced Scorecard perspectives to categorize strategic objectives when OKRs are not available.
- **Source:** Kaplan, R.S. & Norton, D.P. (1996). *The Balanced Scorecard*.

## 2. Alignment Analysis Techniques

### Strategy Decomposition

| Level | Artifact | Example |
|-------|----------|---------|
| Vision | Mission statement | "Become the leading digital bank in LATAM" |
| Strategic Themes | 3-5 investment themes | "Digital Customer Experience" |
| Objectives | Measurable goals per theme | "Reduce customer onboarding time by 60%" |
| Key Results | Metrics per objective | "Digital onboarding completion rate >85%" |
| Projects | Initiatives that deliver key results | "Mobile Onboarding App Development" |

### Alignment Scoring Matrix

| Score | Alignment Level | Description |
|-------|----------------|-------------|
| 5 | Direct | Project directly delivers a Key Result |
| 4 | Strong | Project significantly contributes to an Objective |
| 3 | Moderate | Project supports a Strategic Theme |
| 2 | Indirect | Project enables another project that is aligned |
| 1 | Weak | Project has tenuous connection to strategy |
| 0 | Orphan | No identifiable strategic connection |

### Strategic Orphan Detection

Projects scoring 0-1 on alignment should be:
1. Re-examined for hidden strategic value (regulatory compliance, technical enabler)
2. Re-justified with explicit strategic connection
3. De-prioritized if no justification found
4. Cancelled if resources are needed for higher-alignment projects

### Coverage Gap Detection

Strategic objectives with no supporting projects represent coverage gaps:
- **Critical gap:** Strategic objective with budget allocated but no project
- **Emerging gap:** Strategic objective with declining project support due to cancellations
- **Future gap:** New strategic objective from recent strategy pivot with no project pipeline

## 3. Portfolio Investment Analysis

### Investment vs. Strategy Allocation

Compare percentage of portfolio budget allocated to each strategic theme against the strategic priority ranking:

| Strategic Theme | Priority Rank | Budget Allocation | Alignment |
|----------------|:------------:|:-----------------:|:---------:|
| Digital CX | 1 (highest) | 35% | Aligned |
| Operational Excellence | 2 | 15% | Under-invested |
| Regulatory Compliance | 3 | 40% | Over-invested |
| Innovation | 4 | 10% | Under-invested |

Misalignment >15% between priority rank and budget allocation warrants portfolio rebalancing.

## 4. Drift Detection

### Strategy Drift Indicators

| Indicator | Signal | Action |
|-----------|--------|--------|
| Scope creep away from strategic objectives | Project scope changes that dilute strategic contribution | Reassess alignment score |
| Strategic pivot without project adjustment | Organization changes strategy but projects continue unchanged | Trigger portfolio rebalancing |
| Key Result metric declining despite project progress | Project delivers outputs but not strategic outcomes | Review project approach |
| New executive with different priorities | Leadership change alters strategic emphasis | Refresh alignment analysis |
