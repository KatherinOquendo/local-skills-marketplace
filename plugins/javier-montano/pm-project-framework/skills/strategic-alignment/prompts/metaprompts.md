# Metaprompts — Strategic Alignment & OKR Tracking

> Meta-level strategies to maximize alignment analysis effectiveness.

---

## 1. Orphan Rehabilitation Protocol

**Purpose:** Before recommending project cancellation for orphans, attempt to find legitimate strategic value.

```
For each project scoring ≤1 on alignment:

1. HIDDEN VALUE CHECK:
   - Does this project enable another project that IS aligned?
   - Is this a regulatory/compliance requirement (must-do regardless)?
   - Does this reduce operational risk that protects strategic initiatives?

2. STAKEHOLDER INTENT:
   - Interview the project sponsor: what strategic value did they intend?
   - Is the misalignment a scoring error or a genuine lack of strategic fit?

3. REHABILITATION:
   - Can the project scope be adjusted to increase strategic alignment?
   - Can the project be merged with a strategically aligned initiative?

4. FINAL VERDICT:
   - If rehabilitation fails → recommend cancel/deprioritize
   - If hidden value found → update alignment score with [STAKEHOLDER] evidence
   - Document reasoning either way
```

## 2. Investment Coherence Check

**Purpose:** Verify that budget allocation matches strategic priorities.

```
For each strategic theme:

1. Calculate: % of total portfolio budget allocated
2. Calculate: strategic priority ranking (1 = highest)
3. Compare: is the highest-priority theme getting the most investment?
4. If misaligned by >15%:
   - Is the misalignment intentional? (regulatory mandates, sunk costs)
   - Is it historical inertia? (legacy projects consuming budget)
   - Recommend rebalancing with specific magnitude

Present as a waterfall chart showing current vs. optimal allocation.
Tag all allocation data [METRIC] and recommendations [PLAN].
```
