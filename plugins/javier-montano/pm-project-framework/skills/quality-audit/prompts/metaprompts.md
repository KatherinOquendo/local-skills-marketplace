# Metaprompts — Quality Audit

## Meta-Strategy 1: Auditor Objectivity Check

**Purpose:** Ensure audit findings are evidence-based, not opinion-based

```
For each audit finding, verify:
1. EVIDENCE SUFFICIENCY — Is there at least 2 independent pieces of evidence?
2. CRITERIA CLARITY — Is the audit criterion unambiguous? Could the auditee reasonably disagree?
3. ROOT CAUSE DEPTH — Is the finding addressing a symptom or the actual root cause? Apply 5-Whys.
4. SEVERITY CALIBRATION — Compare severity to similar findings in past audits for consistency.
5. ACTIONABILITY — Can the auditee realistically implement the corrective action?
Reclassify or remove findings that fail 2+ checks.
```

## Meta-Strategy 2: Systemic Pattern Detection

**Purpose:** Identify systemic issues from individual audit findings

```
After compiling individual findings, analyze for patterns:
1. PROCESS CLUSTER — Are multiple findings pointing to the same broken process?
2. TEAM CLUSTER — Do findings concentrate in specific teams? Why?
3. TEMPORAL CLUSTER — Are findings recurring from previous audits? Root cause not addressed.
4. SEVERITY ESCALATION — Are minor findings from last audit becoming major this time?
Synthesize patterns into systemic recommendations beyond individual corrective actions.
```

## Meta-Strategy 3: Positive Finding Amplification

**Purpose:** Balance audit reports with strengths, not just deficiencies

```
For every 3 non-conformances identified, actively seek 1 best practice:
1. What process is working exceptionally well? Document and recommend for replication.
2. What corrective action from a previous audit resulted in measurable improvement?
3. What team innovation improved quality beyond the standard requirement?
Include a "Strengths" section in the audit report alongside findings.
```
