# State of the Art — Cost Estimation (2024-2026)

## Trend 1: ML-Calibrated Estimation Models
Machine learning models trained on historical project data auto-calibrate parametric estimates, improving accuracy by 20-30% over expert judgment alone. [DOC]
- **Tools**: Custom regression models, Forecast.app, estimation databases [DOC]
- **Impact**: Class 4 estimates achieve Class 3 accuracy [METRIC]

## Trend 2: Reference Class Forecasting
Bent Flyvbjerg's reference class forecasting corrects optimism bias by comparing estimates against actual outcomes from similar past projects. [DOC]
- **Method**: Find reference class then calculate distribution then select appropriate percentile [PLAN]
- **Impact**: 40% more accurate than traditional methods [METRIC]

## Trend 3: Probabilistic Estimation
Single-point estimates replaced by probability distributions. Sponsors approve at specific confidence levels (P50 for likely, P80 for budgeting). [DOC]
- **Tools**: Monte Carlo simulation, PERT distributions [METRIC]
- **Adoption**: 45% of mature PMOs use probabilistic estimation [INFERENCIA]

## Trend 4: Story Point to FTE-Month Conversion
Agile teams calibrate story point to FTE-month conversion factors for budget estimation, bridging agile sizing with financial planning. [DOC]
- **Pattern**: Historical velocity x cost-per-sprint = cost-per-story-point [METRIC]
- **Caveat**: Conversion factors vary by team and technology [SUPUESTO]

## Trend 5: Crowd-Sourced Estimation
Planning poker and Wideband Delphi techniques leverage team collective intelligence. Research shows wisdom of crowds outperforms individual expert estimates. [DOC]
- **Evidence**: Group estimates 15% more accurate than individual [METRIC]
- **Condition**: Independent thinking before sharing (avoid anchoring) [INFERENCIA]

## Forecast
By 2026, cost estimation will be AI-assisted with automatic reference class lookup, probabilistic output, and continuous calibration against actuals. [INFERENCIA]
