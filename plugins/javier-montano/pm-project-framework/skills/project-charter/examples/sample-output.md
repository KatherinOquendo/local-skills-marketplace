# 00 Project Charter — Acme Corp CRM Cloud Migration

> **Proyecto:** Acme Corp CRM Cloud Migration | **Fecha:** 10 de marzo de 2026
> **Tipo:** Hybrid | **Sponsor:** Maria Lopez, VP Commercial

---

## 1. Executive Summary

Acme Corp will migrate its on-premises CRM (Salesforce Classic) to Salesforce Lightning on cloud infrastructure, consolidating 3 regional instances into a single global tenant. The migration will improve sales team productivity by 25%, reduce CRM infrastructure costs by 40%, and enable real-time analytics across all regions. [PLAN]

## 4. Scope

**In scope:** Data migration (leads, contacts, opportunities, accounts), workflow automation migration, 12 custom report migrations, user training (3 sessions × 3 regions), post-go-live support (30 days).

**Out of scope:** Marketing automation (separate project Q1 2027), third-party integrations beyond ERP (Phase 2), mobile app customization, historical data older than 5 years.
