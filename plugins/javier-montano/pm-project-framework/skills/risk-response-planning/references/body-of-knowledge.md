# Body of Knowledge — Risk Response Planning

## 1. Definition and Scope

Risk response planning develops strategies and actions to reduce threats and enhance opportunities. For each identified risk, it defines the response strategy, specific actions, responsible owners, triggers, and contingency plans. [DOC]

## 3. Threat Response Strategies

| Strategy | Description | When to Use |
|----------|-------------|-------------|
| Avoid | Eliminate the threat by changing plan | High probability + high impact |
| Transfer | Shift impact to third party (insurance, contract) | Financial/legal risks |
| Mitigate | Reduce probability or impact | Most common strategy |
| Accept (Active) | Establish contingency plan | Low probability risks |
| Accept (Passive) | Acknowledge, no proactive action | Very low severity risks |
| Escalate | Move ownership to program/portfolio level | Beyond project authority |

## 4. Opportunity Response Strategies

| Strategy | Description | When to Use |
|----------|-------------|-------------|
| Exploit | Ensure opportunity is realized | High-value opportunities |
| Share | Partner to capture opportunity | Need external capabilities |
| Enhance | Increase probability or impact | Moderate opportunities |
| Accept | Take advantage if it occurs | Low-effort opportunities |

## 5. Response Planning Process

1. **Prioritize risks** — Focus on highest-exposure risks first [PLAN]
2. **Select strategy** — Choose appropriate response strategy per risk
3. **Design actions** — Define specific, actionable response steps
4. **Assign owners** — One owner per risk response
5. **Define triggers** — Conditions that activate the response
6. **Estimate residual risk** — Risk remaining after response
7. **Plan contingency** — Fallback if primary response fails [PLAN]

## 6. Response Quality Criteria

| Criterion | Test |
|-----------|------|
| Actionable | Can the response be implemented with available resources? |
| Timely | Can it be implemented before the risk materializes? |
| Proportionate | Is the response cost justified by the risk exposure? |
| Owned | Is there a single accountable owner? |
| Measurable | Can response effectiveness be tracked? [METRIC] |

## 7. Key Metrics

| Metric | Formula | Target |
|--------|---------|--------|
| Response coverage | Risks with responses / Total high risks | 100% for High+ |
| Response implementation rate | Actions completed on time / Total actions | ≥ 90% |
| Residual risk ratio | Post-response exposure / Pre-response exposure | ≤ 40% |
| Contingency utilization | Contingency used / Contingency allocated | 50-75% |
