# Metaprompts — Lessons Register

## Metaprompt 1: Lesson Actionability Enhancer

**Strategy**: Transform vague lessons into specific, actionable recommendations.

**Protocol**:
1. Review each lesson for specificity (who, what, when, how)
2. Flag vague lessons ("communicate better", "plan more")
3. Rewrite as specific actions with measurable outcomes
4. Add context about when and how to apply
5. Rate actionability on 1-5 scale before and after

**Activation**:
> "Review each lesson in this register for actionability. Rewrite vague lessons as specific, actionable recommendations. Each lesson should answer: Who should do what differently, when, and how would we measure improvement?"

## Metaprompt 2: Organizational Learning Pattern Detector

**Strategy**: Find systemic patterns across lessons that indicate organizational capability gaps.

**Protocol**:
1. Cluster lessons by root cause, not just category
2. Identify patterns appearing across 3+ projects
3. Trace to organizational capabilities (skills, processes, tools, culture)
4. Prioritize by frequency x impact
5. Propose capability-building initiatives, not just lesson application

**Activation**:
> "Analyze this lessons register for organizational patterns. Which lessons repeat across projects? What organizational capability gaps do they reveal? Propose capability-building initiatives that would prevent entire categories of lessons from recurring."

## Metaprompt 3: Lesson Application Verifier

**Strategy**: Track whether captured lessons are actually being applied in subsequent projects.

**Protocol**:
1. For each validated lesson, identify projects where it should have been applied
2. Check if those projects show evidence of application
3. If not applied, investigate why (awareness, relevance, practicality)
4. Calculate lesson application rate as a PMO health metric
5. Improve dissemination for under-applied lessons

**Activation**:
> "Verify lesson application. For our top 20 validated lessons, check if subsequent relevant projects actually applied them. Calculate the application rate. For under-applied lessons, investigate barriers and improve the dissemination mechanism."
