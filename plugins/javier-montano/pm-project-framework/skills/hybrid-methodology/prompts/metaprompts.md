# Metaprompts — Hybrid Methodology

## Metaprompt 1: Ceremony Value Audit

**Strategy**: Evaluate every ceremony in the hybrid methodology for value-add vs. waste.

**Protocol**:
1. List all ceremonies (meetings, reviews, gates, standups)
2. For each, quantify: attendees x duration x frequency = cost in hours/sprint
3. Assess value delivered: decisions made, risks identified, alignment achieved
4. Calculate value/cost ratio
5. Eliminate or merge ceremonies below threshold

**Activation**:
> "Audit every ceremony in this hybrid methodology. Calculate the cost (person-hours per sprint) and value (decisions/risks/alignment) of each. Eliminate ceremonies with low value/cost ratio and propose merged alternatives."

## Metaprompt 2: Integration Seam Analysis

**Strategy**: Identify and strengthen the seams where predictive and agile practices meet.

**Protocol**:
1. Map all integration points between predictive and agile elements
2. For each seam, identify information flow, format translation, and timing
3. Detect misalignment (e.g., sprint ending mid-phase)
4. Propose synchronization mechanisms
5. Test with a "day in the life" scenario for each role

**Activation**:
> "Analyze the integration seams in this hybrid methodology. For each point where predictive meets agile, verify that information flows correctly, timing aligns, and roles understand their responsibilities on both sides. Propose fixes for any misalignments."

## Metaprompt 3: Anti-Pattern Detection

**Strategy**: Scan the hybrid methodology for known anti-patterns that undermine hybrid effectiveness.

**Protocol**:
1. Check for Water-Scrum-Fall (agile cosmetics on waterfall reality)
2. Check for dual reporting burden
3. Check for governance mismatch (wrong controls for wrong context)
4. Check for tool fragmentation
5. Propose remediation for each detected anti-pattern

**Activation**:
> "Scan this hybrid methodology for anti-patterns: Water-Scrum-Fall, dual reporting, governance mismatch, tool fragmentation, and culture clash. For each detected pattern, explain the symptom, root cause, and remediation."
