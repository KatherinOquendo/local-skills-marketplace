# Metaprompts — PMO Assessment

> PMO-APEX Skill: `pmo-assessment` · Metaprompts para adaptación de evaluaciones
> Última actualización: 2026-03-17

---

## MP-01: Assessment Framework Personalizado

```
Eres el agente de PMO assessment del PMO-APEX. La organización
{nombre_organizacion} en el sector {sector} requiere un framework
de assessment personalizado.
Adapta las 6 dimensiones estándar al contexto del sector:
- Qué dimensiones adicionales son relevantes?
- Qué pesos asignar por dimensión?
- Qué benchmarks sectoriales aplicar?
Genera el framework personalizado con rubrics de evaluación.
```

## MP-02: Assessment para PMO en Crisis

```
El PMO de {nombre_organizacion} está en crisis: {descripcion_crisis}.
Diseña un assessment de emergencia que:
1. Diagnostique las causas raíz en menos de 2 semanas
2. Identifique las 3 acciones de mayor impacto inmediato
3. Establezca métricas de recuperación a 30/60/90 días
4. Defina el mensaje a stakeholders sobre el plan de acción
Prioriza velocidad sobre completitud del assessment.
```

## MP-03: Assessment Comparativo Multi-PMO

```
La organización {nombre_organizacion} tiene {numero} PMOs.
Diseña un assessment comparativo que:
1. Use criterios uniformes para comparación válida
2. Identifique overlaps y gaps de servicio entre PMOs
3. Evalúe sinergias potenciales de consolidación
4. Recomiende modelo target (consolidar, federar, o mantener separados)
Genera matriz comparativa con evidencia por dimensión por PMO.
```

## MP-04: Assessment de PMO Ágil (LACE)

```
La organización {nombre_organizacion} implementa SAFe y tiene
un Lean-Agile Center of Excellence (LACE) además del PMO tradicional.
Diseña un assessment que:
1. Evalúe el LACE contra criterios SAFe
2. Evalúe el PMO contra criterios de governance
3. Identifique solapamientos y conflictos entre LACE y PMO
4. Recomiende modelo de coexistencia o integración
```

## MP-05: PMO Value Story Generator

```
Basándose en los resultados del assessment del PMO de {nombre_organizacion},
genera un "value story" ejecutivo que:
1. Cuantifique el valor del PMO en términos de negocio
2. Compare escenario con PMO vs. sin PMO
3. Proyecte ROI de inversión adicional en PMO
4. Sea presentable a C-level en 5 minutos
Formato: executive briefing con datos, no narración.
```
