# Use Case Prompts — PMO Assessment

> PMO-APEX Skill: `pmo-assessment` · Prompts NL-HP para evaluación de PMO
> Última actualización: 2026-03-17

---

## UC-01: Assessment Completo del PMO

```
Ejecuta una evaluación completa del PMO de {nombre_organizacion}.
Evalúa las 6 dimensiones: alineación estratégica, calidad de governance,
entrega de servicios, satisfacción de stakeholders, gestión de recursos,
y realización de valor. Usa escala 1-5 con evidencia por dimensión.
Genera scorecard y roadmap de transformación.
```

## UC-02: Encuesta de Percepción de Stakeholders

```
Diseña una encuesta de percepción del PMO para {nombre_organizacion}.
Incluye preguntas para 3 audiencias: PMs, sponsors ejecutivos, y equipos
de proyecto. Escala Likert 1-5. Dimensiones: utilidad de servicios,
calidad de reporting, agilidad de respuesta, y valor percibido.
Mínimo 15 preguntas, máximo 25. Incluye instrucciones de aplicación.
```

## UC-03: Service Catalog Maturity Assessment

```
Evalúa la madurez del catálogo de servicios del PMO de {nombre_organizacion}.
Para cada servicio identificado, califica en escala 1-5 (ad-hoc, emergente,
definido, gestionado, optimizado). Calcula adoption rate por servicio.
Identifica servicios faltantes vs. benchmark de industria. Genera
service maturity matrix con recomendaciones de mejora.
```

## UC-04: PMO Value Quantification

```
Cuantifica el valor del PMO de {nombre_organizacion}. Compara métricas
de proyectos con soporte PMO vs. sin soporte: tasa de éxito, on-time
delivery, budget variance, stakeholder satisfaction. Calcula ROI del PMO
como costo del PMO vs. valor generado (cost avoidance + time savings).
Usa evidence tags en toda cuantificación.
```

## UC-05: PMO Operating Model Assessment

```
Clasifica el modelo operativo actual del PMO de {nombre_organizacion}
(supportive, controlling, directive, hybrid). Compara el modelo actual
con las necesidades organizacionales. Identifica mismatches entre modelo
y cultura. Recomienda ajustes al modelo operativo con justificación
basada en madurez PM de la organización.
```

## UC-06: PMO Benchmarking

```
Realiza benchmarking del PMO de {nombre_organizacion} contra estándares
de industria {sector}. Dimensiones: staffing ratio, service catalog breadth,
project success rate, stakeholder satisfaction, methodology coverage.
Identifica gaps significativos y oportunidades de mejora prioritarias.
```

## UC-07: PMO Transformation Roadmap

```
Basándose en los resultados del assessment del PMO de {nombre_organizacion},
diseña un roadmap de transformación en 4 fases:
- Quick wins (0-3 meses)
- Foundation (3-6 meses)
- Maturity (6-12 meses)
- Excellence (12+ meses)
Cada fase con: objetivos, acciones, responsables, métricas de éxito,
y estimación de esfuerzo en FTE-meses.
```

## UC-08: Assessment de PMO Multi-Metodología

```
Evalúa la capacidad del PMO de {nombre_organizacion} para gobernar
un portfolio multi-metodología (Agile, Waterfall, Hybrid).
Verifica: metodologías soportadas, capacidad de coaching por metodología,
governance adaptada por tipo, reporting unificado, y competencias del
equipo PMO por metodología. Identifica gaps de soporte metodológico.
```
