# Metaprompts — PMO Maturity

> PMO-APEX Skill: `pmo-maturity` · Metaprompts para adaptación de evaluaciones
> Última actualización: 2026-03-17

---

## MP-01: Custom Maturity Model

```
Eres el agente de maturity assessment del PMO-APEX. La organización
{nombre_organizacion} en el sector {sector} requiere un modelo de
madurez personalizado porque OPM3/P3M3 no cubren sus necesidades
específicas: {necesidades_especificas}.
Diseña un modelo de madurez custom con:
1. Dimensiones relevantes al sector
2. Escala de 5 niveles con descriptores específicos
3. Método de scoring con rubrics
4. Benchmark calibration approach
```

## MP-02: Maturity for Mergers & Acquisitions

```
{nombre_organizacion} está adquiriendo/fusionándose con {nombre_empresa}.
Diseña un assessment de madurez comparativo que:
1. Evalúe madurez PM de ambas organizaciones con criterios idénticos
2. Identifique gaps de madurez entre las dos organizaciones
3. Evalúe riesgos de integración PM post-merger
4. Diseñe roadmap de armonización de prácticas PM
```

## MP-03: Agile + Traditional Maturity Integration

```
{nombre_organizacion} opera con metodologías mixtas (ágil + waterfall).
Diseña un framework integrado de madurez que:
1. Evalúe madurez waterfall con OPM3/P3M3
2. Evalúe madurez ágil con Agile Fluency / SAFe Assessment
3. Integre ambas evaluaciones en un radar chart unificado
4. Identifique sinergias y conflictos entre ambas madureces
```

## MP-04: Maturity Investment Case

```
Basándote en el assessment de madurez de {nombre_organizacion}
(nivel actual: {nivel_actual}), genera un business case para
inversión en madurez PM:
1. Costo estimado de mejora (FTE-meses por fase)
2. Beneficios esperados (mejora en project success rate, time-to-market)
3. ROI proyectado basado en benchmarks de industria
4. Risk analysis del plan de mejora
Formato: executive brief de 2 páginas.
```

## MP-05: Continuous Maturity Monitoring

```
Diseña un sistema de monitoreo continuo de madurez PM para
{nombre_organizacion} que reemplace el assessment anual discreto:
1. Indicadores proxy medibles mensualmente
2. Dashboard de madurez con auto-actualización
3. Alertas por regresión en dimensiones
4. Cadencia de calibración formal (semestral)
Especifica las fuentes de datos y automatizaciones necesarias.
```
