# Metaprompts — Issue Management

## Metaprompt 1: Systemic Pattern Detector

**Strategy**: Look beyond individual issues to find systemic organizational patterns.

**Protocol**:
1. Cluster issues by root cause, not just category
2. Identify recurring patterns (same type, same phase, same team)
3. Trace patterns to organizational causes (process, skill, tool, culture)
4. Propose systemic fixes rather than individual issue fixes
5. Estimate ROI of systemic fix vs. cost of recurring issues

**Activation**:
> "Analyze the issue register for systemic patterns. Don't just fix individual issues — find the organizational root causes that generate issue clusters. Propose systemic improvements with ROI analysis comparing fix cost to recurring issue cost."

## Metaprompt 2: Escalation Effectiveness Review

**Strategy**: Evaluate whether the escalation process is working or creating bottlenecks.

**Protocol**:
1. Measure average time from escalation trigger to escalation action
2. Measure average time from escalation to resolution
3. Identify issues that should have been escalated but weren't
4. Identify issues that were over-escalated
5. Calibrate escalation thresholds based on findings

**Activation**:
> "Review escalation effectiveness. Measure escalation response times, identify missed escalations and over-escalations, and recalibrate thresholds. Are critical issues reaching the right decision-maker fast enough?"

## Metaprompt 3: Issue Impact Cascade Analysis

**Strategy**: Trace the downstream impact of unresolved issues across project dimensions.

**Protocol**:
1. For each open critical/high issue, map impact on scope, schedule, budget, quality, stakeholders
2. Identify cascade effects (issue A delays task B which blocks team C)
3. Calculate cumulative impact of all open issues
4. Prioritize resolutions by cascade impact, not individual severity
5. Visualize with an impact cascade diagram

**Activation**:
> "Map the cascade impact of all open critical and high issues. How does each unresolved issue affect other project dimensions and downstream activities? Reprioritize resolution order based on cascade impact rather than individual severity."
