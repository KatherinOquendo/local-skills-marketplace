# Body of Knowledge — Scope WBS

## 1. Definition and Scope

The Work Breakdown Structure (WBS) is a hierarchical decomposition of the total scope of work to be carried out by the project team to accomplish the project objectives and create the required deliverables. It organizes and defines the total scope of the project. [DOC]

## 3. WBS Types

| Type | Approach | Best For |
|------|----------|----------|
| Deliverable-based | Organized by project outputs | Most projects (PMI recommended) |
| Phase-based | Organized by project lifecycle phases | Sequential projects |
| Functional | Organized by organizational functions | Matrix organizations |
| Hybrid | Combination of approaches | Complex projects |

## 4. WBS Construction Rules (100% Rule)

The WBS must capture 100% of the work defined by the project scope:
- Every level must represent 100% of the parent element
- No work is excluded (including project management activities)
- No overlap between sibling elements
- Decompose until work packages are estimable (8-80 hour rule) [PLAN]

## 5. WBS Dictionary

Each work package in the WBS should have a dictionary entry containing:
- Work package ID and name
- Description of work
- Responsible person/team
- Acceptance criteria
- Resource requirements
- Duration estimate
- Dependencies [PLAN]

## 6. Scope Statement Components

| Component | Description | Purpose |
|-----------|-------------|---------|
| Product scope | Features and functions of deliverables | Define what is built |
| Project scope | Work required to deliver the product | Define what is done |
| In-scope items | Explicitly included work | Clarity |
| Out-of-scope items | Explicitly excluded work | Boundary protection |
| Assumptions | Conditions assumed true | Risk context |
| Constraints | Limiting factors | Planning context |

## 7. Key Metrics

| Metric | Formula | Target |
|--------|---------|--------|
| Scope stability | Change requests / Baseline items | < 10% per phase |
| WBS completeness | Work packages with dictionary / Total WPs | 100% |
| Decomposition depth | Average WBS levels | 3-5 levels |
| Estimation coverage | Estimated WPs / Total WPs | 100% before baseline |
