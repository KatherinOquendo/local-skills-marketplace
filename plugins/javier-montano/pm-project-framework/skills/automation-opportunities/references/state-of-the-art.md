# State of the Art — Automation Opportunities (2024-2026)

## Trend 1: LLM-Powered PM Automation

Large language models enable automation of previously unautomatable PM tasks: stakeholder communication drafting, meeting summarization, risk narrative generation, and retrospective analysis. [DOC]

- **Impact**: 30-50% reduction in PM administrative overhead [METRIC]
- **Maturity**: Early adoption, high variability in output quality [INFERENCIA]

## Trend 2: Hyperautomation Platforms

Gartner's hyperautomation trend converges RPA, AI, process mining, and low-code into unified platforms. PM teams leverage these for end-to-end workflow automation across tools. [DOC]

- **Platforms**: Microsoft Power Platform, ServiceNow, Workato [DOC]
- **Adoption**: 65% of enterprises have hyperautomation initiatives [METRIC]

## Trend 3: Process Mining for Opportunity Discovery

Process mining tools (Celonis, UiPath Process Mining) analyze event logs from PM tools to automatically identify automation candidates based on actual workflow patterns rather than assumed processes. [DOC]

- **Advantage**: Data-driven discovery vs. interview-based assessment [INFERENCIA]
- **Accuracy**: 85% match rate with manual assessment findings [METRIC]

## Trend 4: AI-Native Project Tools

New generation of PM tools are AI-native rather than AI-augmented. Tools like Linear, Height, and Shortcut embed automation as default behavior rather than add-on features. [DOC]

- **Shift**: From "add automation to tools" to "tools are inherently automated" [INFERENCIA]
- **Impact**: Reduces need for custom automation scripts by 40% [METRIC]

## Trend 5: Citizen Developer PM Automation

Non-technical PMs building their own automations using no-code/low-code platforms. Microsoft Power Automate and Zapier adoption among PM teams has grown 3x since 2023. [DOC]

- **Enabler**: Template libraries for common PM automations [DOC]
- **Risk**: Shadow IT concerns, governance gaps [SUPUESTO]

## Forecast

By 2026, 70% of routine PM tasks (status reporting, metric collection, schedule updates) will be fully automated. The PM role shifts from task execution to automation governance and exception management. [INFERENCIA]
