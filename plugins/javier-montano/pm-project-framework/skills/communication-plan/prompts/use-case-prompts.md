# Use-Case Prompts — Communication Plan

## Prompt 1: Comprehensive Communication Plan

**When**: Project kickoff.

**Context variables**:
- `{stakeholder_register}` — Stakeholders with roles and influence levels
- `{project_phases}` — Project phases and milestones

**Deliver**:
> Create a communication plan for `{stakeholder_register}` across `{project_phases}`. For each stakeholder group: information need, channel, frequency, format, owner, and feedback mechanism. Include communication calendar and escalation matrix.

## Prompt 2: Stakeholder Communication Audit

**When**: Stakeholder engagement declining or mid-project review.

**Context variables**:
- `{current_plan}` — Existing communication plan
- `{engagement_metrics}` — Response rates, meeting attendance, feedback

**Deliver**:
> Audit `{current_plan}` against `{engagement_metrics}`. Identify under-served stakeholders, ineffective channels, and communication gaps. Recommend adjustments with expected engagement improvement.

## Prompt 3: Crisis Communication Protocol

**When**: Project escalation or critical incident.

**Context variables**:
- `{incident}` — Nature of the crisis
- `{affected_stakeholders}` — Stakeholders impacted

**Deliver**:
> Design crisis communication protocol for `{incident}` affecting `{affected_stakeholders}`. Include: immediate notification script, escalation chain, update cadence, FAQ document, and post-crisis communication plan.
