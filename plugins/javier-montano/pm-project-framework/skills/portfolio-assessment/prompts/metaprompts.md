# Metaprompts — Portfolio Assessment

> PMO-APEX Skill: `portfolio-assessment` · Metaprompts para adaptación
> Última actualización: 2026-03-17

---

## MP-01: Strategic Alignment Stress Test

```
Stress-test la alineación estratégica del portafolio de {nombre_organizacion}.
Compara prioridades estratégicas declaradas (con pesos %) contra la
asignación real de presupuesto y recursos por proyecto. Calcula el
alignment score (correlación). Identifica proyectos que consumen recursos
sin justificación estratégica. Recomienda rebalanceo con impacto estimado.
```

## MP-02: Portfolio Assessment para Organización Ágil

```
Adapta el assessment de portafolio para {nombre_organizacion} que
opera con SAFe LPM. Reemplaza dimensiones tradicionales por:
1. Portfolio flow (WIP, throughput, cycle time)
2. Participatory budgeting maturity
3. Guardrails effectiveness
4. Value stream alignment
5. Lean governance quality
Genera scorecard con criterios SAFe-specific.
```

## MP-03: Blind Spot Scanner

```
Escanea el assessment de portafolio de {nombre_organizacion} buscando
blind spots. Compara dimensiones evaluadas contra frameworks comprehensivos.
Verifica si faltan: ESG/sustainability, innovation ratio, technical debt,
team health, customer satisfaction. Agrega dimensiones faltantes y
rebalancea el assessment.
```

## MP-04: Portfolio Assessment para M&A

```
{nombre_organizacion} está adquiriendo {nombre_target}. Diseña un
assessment de portafolio comparativo que:
1. Evalúe ambos portafolios con criterios idénticos
2. Identifique sinergias y conflictos entre portafolios
3. Recomiende portafolio combinado post-merger
4. Estime esfuerzo de integración de governance
```

## MP-05: Quick Portfolio Diagnostic

```
Ejecuta un diagnóstico rápido (1 hora) del portafolio de
{nombre_organizacion}. Usa solo 3 dimensiones prioritarias
(strategic alignment, delivery performance, resource health).
Genera un traffic light summary con top 3 acciones inmediatas.
Este es un triage, no un assessment completo.
```
