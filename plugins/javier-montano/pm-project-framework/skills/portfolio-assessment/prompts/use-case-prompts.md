# Use Case Prompts — Portfolio Assessment

> PMO-APEX Skill: `portfolio-assessment` · Prompts NL-HP para evaluación de portafolio
> Última actualización: 2026-03-17

---

## UC-01: Assessment Completo de Portfolio Management

```
Ejecuta un assessment completo de portfolio management para
{nombre_organizacion}. Evalúa las 6 dimensiones (strategic alignment,
governance, prioritization, resource optimization, risk aggregation,
value realization) en escala 1-5. Genera scorecard, capability
heat map, gap analysis, y improvement roadmap con estimación de esfuerzo.
```

## UC-02: Strategic Alignment Deep Dive

```
Evalúa la alineación estratégica del portafolio de {nombre_organizacion}.
Para cada proyecto activo: score de alignment con objetivos estratégicos,
justificación, y gaps. Calcula: % del portafolio alineado, % sin
linkage, y distribución de inversión por objetivo estratégico.
Genera bubble chart conceptual de alignment vs. investment.
```

## UC-03: Prioritization Model Effectiveness

```
Evalúa la efectividad del modelo de priorización de portafolio de
{nombre_organizacion}. Analiza: criterios usados, pesos, consistencia
de aplicación, sensitivity del ranking, y satisfacción de stakeholders
con el proceso. Compara con best practices de MCDA.
Recomienda mejoras al modelo de scoring.
```

## UC-04: Portfolio Governance Assessment

```
Evalúa la governance de portafolio de {nombre_organizacion}.
Analiza: cuerpos de decisión, frecuencia, participantes, calidad
de decisiones, escalation paths, y reporting cadence.
Score en escala 1-5 con evidencia. Recomienda mejoras al governance
framework con estimación de esfuerzo de implementación.
```

## UC-05: Resource Optimization Assessment

```
Evalúa la gestión de recursos cross-project en {nombre_organizacion}.
Analiza: modelo de asignación, visibilidad cross-project, conflictos
de recursos, utilización, y capacity planning. Identifica cuellos
de botella sistémicos y recomienda modelo de optimización.
```

## UC-06: Benefits Realization Assessment

```
Evalúa la capacidad de realización de beneficios de {nombre_organizacion}.
Analiza: se definen beneficios al inicio?, se miden post-delivery?,
hay tracking continuo?, quién es accountable?, qué % del portafolio
tiene beneficios medidos y realizados? Score en escala 1-5.
```

## UC-07: Lean Portfolio Assessment (SAFe)

```
Evalúa las prácticas de Lean Portfolio Management de {nombre_organizacion}
contra el framework SAFe LPM. Dimensiones: Strategy and Investment
Funding, Agile Portfolio Operations, Lean Governance. Score per
practice area. Genera gap analysis y roadmap de adopción LPM.
```

## UC-08: Portfolio Assessment Executive Briefing

```
Genera un executive briefing del assessment de portafolio de
{nombre_organizacion} para el C-level. Formato: 1 página con
scorecard visual (6 dimensiones), top 3 gaps, investment requerido
para mejora, y ROI esperado. Lenguaje de negocio, no técnico.
```
