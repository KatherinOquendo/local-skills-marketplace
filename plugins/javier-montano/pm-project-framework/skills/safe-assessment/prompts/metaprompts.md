# Metaprompts — SAFe Assessment

## Meta-Strategy 1: Ceremony vs. Value Check

**Purpose:** Ensure assessment measures outcomes, not just process compliance

```
For each SAFe practice assessed, ask:
1. Is the ceremony being performed? (Process compliance)
2. Is the ceremony producing VALUE? (Outcome effectiveness)
3. Would the team notice if the ceremony stopped? (Perceived value)
Score practices on both compliance AND value delivery. High compliance + low value = "SAFe theater" — flag for redesign.
```

## Meta-Strategy 2: Cultural Readiness Validation

**Purpose:** Assess whether cultural foundation supports SAFe adoption

```
Evaluate cultural enablers:
1. PSYCHOLOGICAL SAFETY — Can teams raise problems without fear?
2. LEARNING CULTURE — Are failures treated as learning opportunities?
3. EMPOWERMENT — Do teams have authority to make decisions?
4. TRANSPARENCY — Is information freely shared across levels?
If cultural scores are low, process maturity improvements will not stick. Prioritize culture.
```

## Meta-Strategy 3: Assessment Objectivity Assurance

**Purpose:** Prevent assessment bias from internal politics

```
Check for assessment bias:
1. Are self-reported scores validated against objective metrics?
2. Are assessment results used punitively? (If yes, scores will be inflated)
3. Is the assessor independent from the teams being assessed?
4. Are diverse perspectives included (not just leadership view)?
Triangulate: self-assessment + metrics + external observation for each dimension.
```
