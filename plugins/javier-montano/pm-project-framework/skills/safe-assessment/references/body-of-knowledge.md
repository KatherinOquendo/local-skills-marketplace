# Body of Knowledge — SAFe Assessment

## 1. Definition and Scope

A SAFe (Scaled Agile Framework) assessment evaluates an organization's readiness, maturity, and effectiveness in implementing SAFe practices across teams, ARTs (Agile Release Trains), and the portfolio. It measures alignment to SAFe principles and identifies improvement areas. [DOC]

## 3. Assessment Dimensions

| Dimension | Focus Area | Key Metrics |
|-----------|-----------|-------------|
| Team Agility | Scrum/Kanban maturity, iteration execution | Velocity stability, predictability [METRIC] |
| ART Execution | PI Planning, System Demo, I&A | PI predictability, feature cycle time |
| Continuous Delivery Pipeline | DevOps, release on demand | Deployment frequency, lead time |
| Lean Portfolio Management | Strategy, funding, governance | WSJF usage, portfolio flow |
| Organizational Agility | Business agility, lean thinking | Time-to-market, innovation rate |
| Lean-Agile Leadership | Mindset, coaching, communities | Leader engagement, training completion |

## 4. Maturity Levels

| Level | Description | Indicators |
|-------|-------------|-----------|
| 1 — Initial | Ad hoc practices, no consistency | No ceremonies, no metrics |
| 2 — Emerging | Some teams practicing, inconsistent | Partial adoption, early metrics |
| 3 — Defined | ART-level practices established | PI Planning regular, system demos |
| 4 — Measured | Metrics-driven improvement | Predictability > 80%, flow metrics |
| 5 — Optimized | Continuous improvement embedded | Business agility, innovation pipeline |

## 5. Assessment Process

1. **Scope definition** — Determine assessment boundaries (team, ART, portfolio)
2. **Data collection** — Surveys, interviews, artifact review, metric analysis [METRIC]
3. **Analysis** — Score each dimension against maturity model
4. **Gap identification** — Current vs. target maturity gaps
5. **Roadmap creation** — Prioritized improvement plan
6. **Review and approval** — Stakeholder alignment on findings [STAKEHOLDER]

## 6. Key Assessment Tools

- SAFe Business Agility Assessment (official tool)
- Team self-assessment surveys
- PI predictability tracking
- DevOps health radar
- Lean Portfolio Management maturity matrix [PLAN]
