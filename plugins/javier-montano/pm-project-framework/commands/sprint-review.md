# PMO-APEX · SPRINT REVIEW FACILITATION · NL-HP v3.0

## ROLE

Sprint Review Facilitator — activates `apex-ceremony-facilitation` as primary skill. Review mode: 60 minutes.

## AUTO-DETECTION PROTOCOL

1. **Project root**: Use current working directory.
2. **Sprint context**: Scan for completed user stories, demo-ready features, sprint goal.
3. **Stakeholder list**: Load from 01_Stakeholders for review attendees.

## PROTOCOL

1. **Sprint Goal Review** — sprint goal achieved? Partially? Not achieved? Evidence. [DOC]
2. **Demo Agenda** — features to demonstrate, order, presenter, time allocation. [DOC]
3. **Completed Items** — table: item, acceptance criteria, status (done/partial/not done), demo ready. [DOC]
4. **Stakeholder Feedback Capture** — structured format: feature, feedback, priority, action needed. [STAKEHOLDER]
5. **Backlog Adaptation** — items to add, reprioritize, or remove based on feedback. [DOC] [STAKEHOLDER]
6. **Velocity Update** — points completed, velocity trend, forecast for next sprint. [INFERENCIA]
7. **Release Burndown Update** — progress toward release goal. On track / at risk. [INFERENCIA]
8. **Next Sprint Preview** — proposed sprint goal and key items. [INFERENCIA]
