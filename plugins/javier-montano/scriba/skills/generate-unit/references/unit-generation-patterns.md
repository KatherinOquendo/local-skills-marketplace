# Unit Generation Patterns — Reference for generate-unit

> Extracted from rag-estructura-editorial.md: the 4-phase model, section components, Sinapsis conventions.

## Sinapsis Constructivist Model

The Sinapsis editorial line follows a constructivist approach with neuroeducational integration:

- **Pedagogical model**: Experiencia > Conceptualizacion > Reflexion > Aplicacion
- **Neuroeducational integration**: Emotional recognition, metacognition, self-awareness
- **4 Delors Pillars**: Learn to know, learn to do, learn to live together, learn to be
- **Cooperative learning**: Cognitive development through peer collaboration

## Unit Anatomy — 4 Sections

### 1. Apertura (Opening)

| Component | Purpose | Taxonomy Level |
|-----------|---------|---------------|
| Unit title + module list | Orientation, advance organizer | — |
| Metacognitive questions | Activate prior knowledge | Recordar |
| ODS link (icon + number) | Curricular insertion INS-DS | — |
| "Me preparo para mis competencias" | List competencies to develop | — |
| "Seguridad Integral" | Safety tips (INS-SI) | — |
| "Piensa y Reflexiona" | Prior knowledge activation | Recordar/Recuperacion |

### 2. Desarrollo (Development) — Per Theme

Each theme cycles through 4 phases:

| Phase | Function | Bloom | Marzano |
|-------|----------|-------|---------|
| Experiencia | Connect to lived experience | Recordar | Recuperacion |
| Conceptualizacion | Build new concept | Comprender | Comprension |
| Reflexion | Critical analysis, connections | Analizar | Analisis |
| Aplicacion | Practice, projects, productions | Aplicar-Crear | Utilizacion |

### 3. Cierre (Closing)

| Component | Function | Taxonomy |
|-----------|----------|----------|
| "Construyo mi Aprendizaje" | Autonomous synthesis | Crear/Metacognicion |
| "Mi emocion cuenta" | Emotional connection (INS-SE) | Sistema del Yo |
| "Ponte pilas" | Attention reactivation | Engagement reset |

### 4. Evaluacion (Assessment)

| Type | When | Format |
|------|------|--------|
| Diagnostica | Unit opening | Open questions, classification exercises |
| Formativa | During desarrollo | "Construyo mi Aprendizaje", "Ponte pilas" |
| Sumativa | Unit closing | Rubric-based, 3+ levels, DZ-linked |

## Traceability Convention

Every content paragraph carries a tag:

```
[REF: DZ-<ASIG>-<ANO>-<ID>]
```

Example: `[REF: DZ-LENG-5-001]` — This paragraph develops the skill of identifying narrative elements.

## Mandatory Insertions Per Unit

- INS-DS (ODS): At least 1 reference to Sustainable Development Goals
- INS-SE (Socioemocional): "Mi emocion cuenta" section
- INS-SI (Seguridad Integral): Safety tip in apertura
- INS-CEI (Civica-Etica): At least 1 values connection

## Vocabulary Constraints (EGB Media 5to-7mo)

| Parameter | Limit | Rationale |
|-----------|-------|-----------|
| Words per sentence | Max 15-20 | Working memory capacity at age 9-12 |
| Average syllables per word | Max 3 | Fluent decoding threshold |
| New concepts per paragraph | Max 1-2 | Avoid intrinsic overload |
| Technical terms per page | Max 3-5 | With mandatory contextual definition |

## Sinapsis Voice Rules (Summary)

- Active voice preferred over passive
- Direct address to student ("Observa...", "Reflexiona...")
- Inclusive language, free of stereotypes
- Concrete before abstract
- Examples from Ecuadorian context when possible
