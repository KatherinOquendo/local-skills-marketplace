# Guide Evolution Patterns

> Domain knowledge for the `evolve-guide` skill.

## Guide Improvement Checklist

Run this checklist to assess the current state of a teacher guide before applying feedback.

### Structural Completeness

- [ ] Planning section present with lesson timing
- [ ] Material/resource list complete
- [ ] Per-activity methodology suggestions present
- [ ] DUA adaptations section present per activity
- [ ] Rubric(s) present with all required indicators
- [ ] Answer key / expected responses included
- [ ] Assessment guidance present (formative + summative)
- [ ] Differentiation suggestions for advanced/struggling students

### DUA Baseline Assessment

For each activity, count current adaptations:

| Activity | Representation | Engagement | Action/Expression | Total | Meets Threshold? |
|----------|---------------|------------|-------------------|-------|-----------------|
| Act X.Y | count | count | count | sum | Yes/No (>=2) |

### Rubric Quality Assessment

For each rubric, verify:

- [ ] All indicators from Matriz A&S represented
- [ ] Each level has distinct, observable descriptors
- [ ] No vague language ("partially", "somewhat", "adequately")
- [ ] Levels are progressively ordered
- [ ] Level descriptors use student-observable behaviors

### Alignment Check

- [ ] Guide activities match companion unit activities
- [ ] Guide references match unit content
- [ ] Rubric indicators match unit destrezas
- [ ] Timing suggestions are realistic for grade level

## DUA Enhancement Patterns

### Representation Adaptations (How students access content)

| Pattern | Description | Implementation Example | Grade Suitability |
|---------|-------------|----------------------|-------------------|
| Visual organizer | Graphic representation of concept relationships | "Proporcione un organizador grafico tipo mapa conceptual con los conceptos clave pre-llenados" | All |
| Multi-modal text | Same content in text + image + audio | "Lea el texto en voz alta mientras los estudiantes siguen con la lectura. Senale las imagenes correspondientes." | 5to-6to |
| Vocabulary scaffold | Key terms with visual definitions | "Entregue tarjetas de vocabulario con la palabra, definicion simple, e imagen representativa" | All |
| Concrete manipulatives | Physical objects representing abstract concepts | "Use bloques o recortes para representar las fracciones antes de pasar al formato numerico" | 5to |
| Bilingual support | Key terms in Spanish + indigenous language | "Incluya el termino en kichwa cuando sea pertinente para estudiantes bilingues" | Context-dependent |

### Engagement Adaptations (How students stay motivated)

| Pattern | Description | Implementation Example |
|---------|-------------|----------------------|
| Choice board | Students select from activity options | "Ofrezca tres opciones: escribir un parrafo, crear un dibujo con leyenda, o grabar una explicacion oral" |
| Collaborative option | Pair/group alternative to individual work | "Permita que los estudiantes trabajen en parejas para la actividad de clasificacion" |
| Real-world connection | Link to student's daily life | "Pida a los estudiantes que identifiquen ecosistemas en su comunidad antes de la lectura" |
| Self-assessment | Student monitors own progress | "Entregue una lista de cotejo donde el estudiante marque los pasos completados" |
| Gamification element | Game-like structure | "Organice la actividad como una busqueda del tesoro de conceptos" |

### Action/Expression Adaptations (How students show learning)

| Pattern | Description | Implementation Example |
|---------|-------------|----------------------|
| Multi-format response | Accept different output types | "El estudiante puede responder por escrito, oralmente, o mediante un dibujo rotulado" |
| Template/scaffold | Structured response support | "Proporcione un organizador de parrafos con inicio, desarrollo y cierre pre-estructurado" |
| Technology option | Digital tool alternative | "Permita el uso de herramientas digitales para crear presentaciones como alternativa al poster fisico" |
| Physical demonstration | Kinesthetic response | "El estudiante puede demostrar el concepto mediante una representacion dramatica" |
| Portfolio accumulation | Ongoing evidence collection | "Anada el trabajo a la carpeta de evidencias del estudiante para revision acumulativa" |

### Threshold Enforcement

The minimum is 2 adaptations across at least 2 of the 3 DUA principles per activity. Preferred distribution:

| Compliance Level | Distribution | Assessment |
|-----------------|-------------|-----------|
| Minimum (acceptable) | 1R + 1E, or 1R + 1A, or 1E + 1A | Meets threshold |
| Good | 1R + 1E + 1A | Covers all 3 principles |
| Excellent | 2R + 1E + 1A or 1R + 2E + 1A, etc. | Multiple options per principle |

## Rubric Refinement Workflow

### Step 1: Inventory Current Rubric State

List all indicators, their rubric levels, and current descriptors.

### Step 2: Identify Vague Descriptors

Vague descriptors contain these patterns:

| Vague Pattern | Why It Fails | Replacement Strategy |
|--------------|-------------|---------------------|
| "Logra parcialmente" | Not observable — what does "partial" look like? | Specify count or quality threshold |
| "Demuestra comprension" | Circular — uses the assessed skill in the descriptor | Describe the observable behavior |
| "Adecuadamente" | Subjective — no shared definition of "adequate" | Define the measurable criterion |
| "Con ayuda del docente" | Only applicable to Inicio level; imprecise at other levels | Specify what type of help and how much |
| "De manera satisfactoria" | Undefined satisfaction threshold | Replace with specific evidence |

### Step 3: Write Observable Descriptors

For each level, the descriptor must answer: "What would I see the student doing?"

| Level | Descriptor Formula | Example (Reading Comprehension) |
|-------|-------------------|--------------------------------|
| Inicio | [Verb] + [fewer than N] + [specific elements] | "Identifica menos de 2 ideas principales del texto" |
| En proceso | [Verb] + [N-M] + [specific elements] + [with limitation] | "Identifica 2-3 ideas principales pero no las conecta entre si" |
| Adquirido | [Verb] + [M+] + [specific elements] + [quality indicator] | "Identifica 4+ ideas principales y explica su relacion con ejemplos del texto" |

### Step 4: Verify Indicator Alignment

After refining descriptors, verify:

- Each indicator from the Matriz A&S has a rubric criterion.
- Each rubric criterion maps to at least one activity in the guide.
- Level descriptors align with the taxonomy level of the destreza.

## Change Priority by Dimension

| Priority | Dimension | Rationale |
|----------|-----------|-----------|
| 1 | DUA | Directly affects inclusion — students cannot learn without access |
| 2 | RUBRIC | Affects assessment validity — teachers cannot evaluate without clear criteria |
| 3 | METHODOLOGY | Affects instruction quality — teachers need actionable strategies |
| 4 | PLANNING | Affects logistics — timing and materials support implementation |

## Common Guide Evolution Mistakes

| Mistake | Consequence | Prevention |
|---------|-------------|-----------|
| Adding DUA placeholders instead of real adaptations | Guide appears compliant but provides no practical support | Every adaptation must name a specific resource or strategy |
| Copying rubric descriptors across levels | Teachers cannot distinguish performance levels | Each level must have unique observable behaviors |
| Evolving guide without checking companion unit | Guide references activities that do not exist | Always read both documents before editing |
| Removing indicators during rubric revision | Assessment no longer covers required curriculum | Run indicator coverage verification after every rubric change |
| Ignoring methodology dimension | Teachers get what to teach but not how | Include concrete instructional strategies, not just content |
