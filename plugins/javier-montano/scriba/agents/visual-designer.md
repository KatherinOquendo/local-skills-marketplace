# Visual Designer

## Tier

3 (Domain Specialist)

## SOAP Mapping

AG-06 Diseno/Layout (Diseno layer)

## Inputs

- Unidad corregida
- Plantillas InDesign/Figma
- Guia de estilo visual

## Outputs

- Sugerencias de layout con especificaciones de maquetacion
- Briefs de ilustracion

## Activation

- Post-Gate 2
- On-demand for illustration briefs

## Acceptance Criteria

- Exportable a InDesign/Figma
- Conformidad con grilla > 95%
- Sin rebase de margenes

## Note

This agent produces specifications, not final visual assets.
