# Reglas del Manual de Voz Sinapsis

> Guia de estilo editorial para la linea de textos escolares Sinapsis de Editorial Prolipa [METODOLOGIA]

## Principios de Voz

### Tono General
- **Cercano pero respetuoso**: Hablar al estudiante como un companero de aprendizaje, no como autoridad distante [METODOLOGIA]
- **Motivador sin ser condescendiente**: Estimular la curiosidad sin subestimar la capacidad del lector [INFERENCIA]
- **Inclusivo**: Lenguaje que respeta diversidad cultural, etnica, de genero y capacidades [METODOLOGIA]
- **Constructivista**: El estudiante construye conocimiento, no lo recibe pasivamente [PEDAGOGIA]

### Registro Linguistico por Grado

| Grado | Edad | Oraciones | Vocabulario | Complejidad |
|-------|------|-----------|-------------|-------------|
| 5to EGB | 9-10 | Max 15-20 palabras | Concreto, cotidiano, tecnico guiado | Baja-media |
| 6to EGB | 10-11 | Max 20-25 palabras | Concreto + abstracto introductorio | Media |
| 7mo EGB | 11-12 | Max 25-30 palabras | Abstracto emergente, tecnico con contexto | Media-alta |

[INFERENCIA] Umbrales derivados de los parametros del Student Simulator (nino de 10 anos, 5to EGB)

## Reglas Ortotipograficas

1. **Comas**: Usar coma Oxford cuando la lista tiene 3+ elementos [SUPUESTO]
2. **Negritas**: Solo para terminos clave introducidos por primera vez
3. **Cursivas**: Para titulos de obras literarias y extranjerismos
4. **Comillas**: Dobles para citas textuales, simples para significados
5. **Numeros**: Texto hasta diez, cifra desde 11 en adelante
6. **Guiones**: Largo (—) para incisos, corto (-) para rango numerico

## Estructura de Instrucciones

### Para actividades individuales:
```
[Verbo en imperativo] + [objeto] + [contexto/recurso]
Ejemplo: "Lee el siguiente texto y subraya las palabras que no conozcas."
```

### Para actividades grupales:
```
"En [grupo/pareja], [verbo] + [objeto] + [proposito]"
Ejemplo: "En parejas, comparen sus respuestas y argumenten sus diferencias."
```

### Para reflexion metacognitiva:
```
"[Pregunta abierta que invite a la introspeccion]"
Ejemplo: "¿Que estrategia usaste para resolver este problema? ¿Funcionaria con otros textos?"
```

## Prohibiciones

- NO usar lenguaje sexista o excluyente
- NO incluir estereotipos culturales, etnicos o de genero
- NO usar sarcasmo, ironia sofisticada o humor que pueda confundir
- NO asumir contexto socioeconomico especifico
- NO utilizar regionalismos que excluyan a poblaciones diversas de Ecuador
- NO incluir contenido violento, sexual o inapropiado para la edad
