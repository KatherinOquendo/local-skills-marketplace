## Usage

These examples demonstrate the full output of the `sofka-quality-engineering` skill with `{VARIANTE}=tecnica` and `{FORMATO}=dual`.

**Scenario:** Acme Corp, a mid-size fintech with a monolithic Java backend migrating to microservices. Current maturity Level 2 (Repeatable). Target: Level 4 (Managed) in 12 months.

## Key Sections Demonstrated

1. **S1: Quality Maturity Assessment** — 5-level model scoring, 6 assessment dimensions, gap analysis
2. **S2: Test Strategy** — Architecture-driven test pyramid with ratio targets
3. **S3: Automation Architecture** — Framework selection, CI/CD pipeline stages
4. **S4: Quality Gates** — 5-stage gate criteria with pass/fail and escalation
5. **S5: Quality Metrics** — Leading and lagging indicators with targets
6. **S6: Implementation Plan** — Phased 12-week plan with success criteria
