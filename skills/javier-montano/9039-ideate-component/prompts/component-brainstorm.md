# Prompt: Component Ideation

```
Brainstorm a {COMPONENT_TYPE} for the plugin {PLUGIN_NAME}.

User idea: {IDEA}

Generate a concept card with:
- Name (kebab-case)
- Type (skill, agent, command, hook)
- Movement assignment (1-9)
- Purpose (1-2 sentences)
- Trigger phrases (for skills/commands)
- Inputs and outputs
- Tools needed
- MOAT depth recommendation (Light/Medium/Deep)
- Dependencies on other components
- Complexity estimate (Low/Medium/High)
- Differentiation (what makes it unique)
- Risks and edge cases

If the idea is vague, propose 2-3 concrete variations for the user to choose from.
```
