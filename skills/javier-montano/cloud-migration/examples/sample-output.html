A-01: Cloud Migration Plan — Acme Corp Banking Modernization

Cliente: Acme Corp | Fecha: 12 de marzo de 2026 | Variante: Técnica

Resumen Ejecutivo

Acme Corp debe migrar 42 aplicaciones bancarias desde dos datacenters on-premises (Dallas, Miami) a AWS antes de octubre 2027. El portfolio incluye core banking (Oracle/AIX), 12 microservicios Java, procesamiento de tarjetas (ISO 8583), y sistemas regulatorios.

Conclusiones y Recomendaciones

  1. Iniciar Wave 0 inmediatamente — Direct Connect tiene 4-8 semanas de lead time.
  2. No refactorear Core Banking durante migración — replatform primero, refactor en Year 2.
  3. Discovery con Cloudamize 30+ días antes de Wave 2 para capturar patrones mensuales.
  4. Retirar 7 apps antes de Wave 3 — cada migración innecesaria cuesta $15K-40K.
  5. Presupuestar $1.8M para dual-run — item más subestimado en programas de migración.
