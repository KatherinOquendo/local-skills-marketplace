Guardrails de Producción

  • Kafka: Monitorear consumer lag, replication factor, ISR shrink. Alertar si lag >5 min.
  • Kubernetes: Monitorear pod restart rate, node pressure, HPA scaling. Alertar si restart >3/hora.
  • AI Fraud: Monitorear accuracy drift semanal. Alertar si F1 cae >2 puntos. Revisión humana >$10K.

Vendor Exit Strategy

  • Vendor X AI: Si PoC falla: pipeline open-source con Scikit-learn + feature engineering + MLflow. Estimado: 3-4 sprints. Mantener interfaz abstracta desde día 1.
  • EKS: Si costos exceden presupuesto: ECS Fargate como fallback (menor complejidad, menor flexibilidad).

Re-evaluation Triggers

  • Vendor X recibe nueva ronda de funding o es adquirido → re-evaluar vendor risk
  • Equipo K8s completa programa de capacitación → re-evaluar de 🟡 a 🟢
  • Nueva regulación de AI en sector financiero → re-evaluar governance framework
  • Cualquier tecnología cambia major version → re-evaluar maturity
