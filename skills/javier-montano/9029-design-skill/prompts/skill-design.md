# Prompt: Detailed Skill Design

```
Design the skill {SKILL_NAME} for plugin {PLUGIN_NAME}.

Produce a complete design document with:
1. Purpose and trigger description
2. Tools needed (allowed-tools list)
3. Step-by-step procedure (numbered, tool-aware, input/output per step)
4. Quality criteria (5+ measurable, testable)
5. Anti-patterns (4+ with WHY explanations)
6. Edge cases (4+ boundary conditions)
7. MOAT asset plan: which references, examples, and prompts to create
8. Integration points with other skills in the plugin

The procedure must be specific enough that another agent can execute it without ambiguity.
Each quality criterion must be verifiable with a yes/no check.
```
