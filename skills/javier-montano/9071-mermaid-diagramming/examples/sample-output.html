Figure 3: Loan Origination — Actor Interactions

Sequence diagram showing the interaction between customer, loan officer, credit engine, and approval committee during loan origination.

sequenceDiagram
    actor Customer
    participant LoanOfficer as Loan Officer
    participant CreditEngine as Credit Engine
    participant Committee as Approval Committee
    participant CoreSystem as Core Banking

    Customer->>LoanOfficer: Submit loan application
    LoanOfficer->>CreditEngine: Request credit assessment
    CreditEngine->>CreditEngine: Run scoring model
    CreditEngine-->>LoanOfficer: Credit score + risk tier

    alt Risk tier = Low
        LoanOfficer->>CoreSystem: Auto-approve + create loan
        CoreSystem-->>Customer: Approval notification
    else Risk tier = Medium
        LoanOfficer->>Committee: Escalate for review
        Committee->>Committee: Evaluate collateral + history
        Committee-->>LoanOfficer: Decision (approve/reject)
        LoanOfficer-->>Customer: Decision notification
    else Risk tier = High
        LoanOfficer-->>Customer: Decline with explanation
    end

    Note over CoreSystem: Loan created in core system with amortization schedule
