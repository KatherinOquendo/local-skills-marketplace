Compensation Table

StepActionCompensationTimeout
1Fraud screeningNo compensation needed5s

Outbox Pattern with Debezium CDC All services write events to an outbox table in the same DB transaction as the domain change. Debezium reads the PostgreSQL WAL and streams outbox rows to Kafka with near-real-time latency.
