# Validate Naming

> "Naming is the first act of architecture -- get it wrong and everything built on top inherits the confusion."

Scans all files and directories in a plugin tree and validates that names conform to kebab-case convention (`^[a-z][a-z0-9]*(-[a-z0-9]+)*$`). Reports each violation with its location and a suggested corrected name.

---

## Procedure

### Step 1 -- Collect All Paths

- Use `[Glob]` to list all files and directories under the plugin root.
- Exclude `.git/`, `node_modules/`, and other VCS/dependency directories.
- Store the list for analysis.

### Step 2 -- Extract Base Names

- For each path, extract the base name (file name without extension, or directory name).
- Skip dotfiles (`.gitignore`, `.claude-plugin`) -- these follow their own conventions.

### Step 3 -- Validate Against Kebab-Case Pattern

- Test each base name against: `^[a-z][a-z0-9]*(-[a-z0-9]+)*$`.
- For violations, generate a suggested correction by: lowercasing, replacing spaces/underscores with hyphens, removing special characters.

### Step 4 -- Generate Report

- Output a findings table: `| Severity | Path | Current Name | Suggested Name |`.
- All naming violations are `WARNING` severity.
- Include summary: total files scanned, violations found, pass rate.

---

## Quality Criteria

1. Every violation includes the full relative path from plugin root.
2. Every violation includes a concrete suggested correction.
3. Dotfiles and VCS directories are excluded from checks.
4. The report includes a summary with total scanned and violation count.

## Anti-Patterns

1. Reporting dotfiles as naming violations -- `.gitignore` and `.claude-plugin` are valid names outside kebab-case scope.
2. Suggesting corrections that collide with existing names -- always check for conflicts.
3. Marking SKILL.md as a violation -- uppercase filenames required by convention are exempt.

## Edge Cases

1. Plugin with all-valid names -- report zero violations with a congratulatory summary.
2. Name with consecutive hyphens (`my--skill`) -- invalid kebab-case. Suggest `my-skill`.
