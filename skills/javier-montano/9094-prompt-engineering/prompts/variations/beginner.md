# Simple Prompt Design

Tell the AI: WHO it is, WHAT you need, and HOW to format the answer.

Template:
```
You are a [role].
I need [what you want].
Please format your answer as [format].
```

Example:
```
You are a marketing copywriter.
I need a tagline for a productivity app aimed at freelancers.
Please format your answer as 3 options, each one line.
```
