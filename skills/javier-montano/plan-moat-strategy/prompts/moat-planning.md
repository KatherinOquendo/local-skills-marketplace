# Prompt: MOAT Strategy Planning

```
Plan the MOAT asset allocation for plugin {PLUGIN_NAME} with {N} skills.

For each skill, determine:
- Depth: Deep (refs+examples+prompts), Medium (refs+examples), Light (refs only)
- Specific reference files needed
- Number of example files (input/output pairs)
- Whether a reusable prompt template adds value

Criteria for depth assignment:
- Deep: Core skills with high variability, quality gates, complex decisions
- Medium: Standard operations with predictable patterns
- Light: Simple wrappers, utilities, infrastructure

Output table: | Skill | Depth | References | Examples | Prompts | Rationale |
Include budget summary (total files, estimated effort) and priority order.
```
