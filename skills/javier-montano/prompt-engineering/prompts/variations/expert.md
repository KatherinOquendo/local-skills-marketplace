# Advanced Prompt Architecture

For complex tasks requiring multi-step reasoning, self-correction,
or multi-agent coordination.

Patterns covered:
1. Prompt chains with error handling between steps
2. Constitutional self-correction loops
3. Multi-agent prompt routing (different prompts per agent role)
4. Dynamic few-shot selection based on input similarity
5. Structured output with JSON schema validation
