# Excellence Loop — Use Case Prompts

## Prompt 1: Score a Technical Proposal
> Apply the 10-point excellence rubric to this architecture proposal. Score each criterion (completeness, accuracy, evidence, structure, actionability, clarity, diagrams, consistency, audience-fit, next-steps) on a 1-10 scale with specific justification. Identify the top 3 improvement areas and suggest specific changes to raise them above the 7/10 threshold.

## Prompt 2: Iterative Document Refinement
> I have a draft API design document that scored 6/10 on evidence and 5/10 on diagrams in the last review. Apply targeted improvements to raise both criteria above 7/10: add evidence tags to all claims, include a sequence diagram for the authentication flow, and add an API endpoint table. Then re-score to verify the improvements.

## Prompt 3: Establish Team Quality Standards
> Design an excellence rubric customized for our team's deliverables. We produce: technical RFCs, API specifications, runbooks, and incident post-mortems. Create a rubric for each artifact type with: criteria names, descriptions, scoring scale, minimum thresholds, and examples of what a 4/10, 7/10, and 10/10 looks like for each criterion.
