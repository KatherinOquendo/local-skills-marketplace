# Executive Pitch — Examples

Sample outputs for the `executive-pitch` skill.

## Files

| File | Description |
|------|-------------|
| `sample-output.md` | CEO-targeted pitch for Acme Corp Banking digital transformation. Full 7-section variante tecnica. |
| `sample-output.html` | Same pitch rendered as branded HTML. |

## Usage

These examples demonstrate:
- Hero KPIs with quantified metrics
- Problem statement with cost of inaction ($2.4M/yr)
- 4-pillar value proposition with Year 1 impacts
- 3-option comparison with NPV/IRR analysis
- Investment summary ($3.8M over 18 months)
- Call to action with decision deadline and delay cost

## Notes

- All financial figures are illustrative. Real pitches cite sources from prior discovery phases.
- Audience: CEO. For CFO/CTO/Board variants, the conditional logic reshapes emphasis per the skill spec.
