## About the Sample

**Client:** Acme Corp (fictional)
**Engagement:** $3.8M digital transformation — legacy ERP modernization
**Scenario:** Full pipeline (Phases 0-5b complete, Gate 3 approved)
**Variant:** `técnica` (full 8 sections)

The sample demonstrates:
- Executive summary with quantified outcomes
- Commercial activation package with 5-phase pricing
- Operational readiness checklist (team, infra, access, docs)
- 90-day Sprint 0 + Sprint 1-3 kickoff plan
- Governance transition from discovery to execution roles
- Risk tracker with 5 inherited risks and early warning indicators
- Stakeholder transition matrix (8 stakeholders)
- Mermaid diagrams: governance flow, gantt, state machine
