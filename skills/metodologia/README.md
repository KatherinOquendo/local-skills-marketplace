# MetodologIA Skills (Mirror)

This directory mirrors MetodologIA-branded skills from the MAO (MetodologIA Agentic Orchestrator) plugin framework.

## License

These skills are licensed under **GPL-3.0**, separate from the MIT license that covers the rest of this repository.

## Upstream Source

The canonical source for MetodologIA skills is the `metodologia-discovery-framework` plugin. This directory is a development/reference copy.

## Sync Policy

Changes to MetodologIA skills should be made in the upstream plugin first, then mirrored here. This directory is not the source of truth — it's a convenience copy for cross-brand development.
