## Usage

These examples demonstrate the full output of the `metodologia-user-representative` skill with `{VARIANTE}=tecnica` and `{FORMATO}=dual`.

**Scenario:** Review of 3 Acme Corp deliverables — an architecture deep-dive document, an executive pitch, and a cost estimation report. Target audience: mixed (CTO, VP Engineering, Finance Director).

## Key Sections Demonstrated

1. **5-Dimension Scorecard** — Comprehension, Cognitive Load, Accessibility, Adoption Risks, Biases
2. **Top 5 Micro-Adjustments** — Specific copy, structure, and visual changes prioritized by impact
3. **Adoption Risk Assessment** — Stakeholder resistance points with mitigation strategies
4. **Bias Flags** — Technical, organizational, and optimism biases with fixes
5. **Verdict** — CONDITIONAL with explicit next steps for each deliverable
