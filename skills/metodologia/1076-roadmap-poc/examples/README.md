# Roadmap & PoC — Examples

## Contents

| File | Format | Description |
|------|--------|-------------|
| `sample-output.md` | Markdown | Acme Corp PoC roadmap — 6-sprint plan, prerequisites checklist, gate criteria per sprint, team composition, budget breakdown, risk register |
| `sample-output.html` | HTML | Same content rendered as branded MetodologIA HTML deliverable |

## Usage

These examples demonstrate the full output of the `metodologia-roadmap-poc` skill with `{VARIANTE}=tecnica` and `{FORMATO}=dual`.

**Scenario:** Acme Corp Loan Origination PoC — 12-week engagement (6 two-week sprints), Provider+Client hybrid team of 8, budget range $180K-$220K. Goal: validate end-to-end loan processing flow with 3 integrations.

## Key Sections Demonstrated

1. **S1: Pre-Service Kickoff** — Day-by-day Week 0 agenda, parallel setup grid
2. **S2: Prerequisites** — 12 prerequisites with status, owner, deadline, blocker flags
3. **S3: Sprint Breakdown** — 6 sprints with daily task allocation and acceptance criteria
4. **S4: Team & Budget** — 8 FTE allocation, $200K +/- 20% with burn rate
5. **S5: Timeline** — Mermaid Gantt chart with critical path
6. **S6: Gate Criteria** — 7 gates (0>1 through 6>Production) with measurable criteria
7. **S7: Risk Register** — 6 risks with probability, impact, delay days, cost impact, mitigation
