# Dynamic SME — Use Case Prompts

## Prompt 1 — Database Performance SME

> Activate database performance expertise to review the PostgreSQL schema and
> query patterns for an analytics workload. Assess indexing strategy, query
> execution plans, connection pool configuration, and partitioning decisions.
> Use database-domain frameworks and recommend database-specific optimizations.

## Prompt 2 — Security Architecture SME

> Act as a security architecture SME to review the authentication and authorization
> layer of a SaaS platform. Apply OWASP frameworks, assess the OAuth2 implementation,
> review token lifecycle management, and evaluate the API gateway security posture.
> Flag findings with security-domain severity levels.

## Prompt 3 — Event-Driven Architecture SME

> Activate event-driven architecture expertise to assess a Kafka-based event
> sourcing implementation. Review topic design, consumer group strategy,
> idempotency handling, schema evolution approach, and exactly-once semantics.
> Apply domain-specific patterns from Enterprise Integration Patterns and
> flag anti-patterns with domain-appropriate severity.
